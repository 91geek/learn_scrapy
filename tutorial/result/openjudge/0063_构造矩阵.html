0063:构造矩阵

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
20000ms
单个测试点时间限制:
2000ms
内存限制:
256000kB
描述

给定m×n的非负数矩阵a，求另一个m×n的非负数矩阵b，使得

最小，且bij∈[L,R]。


输入
第一行两个数m、n，为矩阵大小。
接下来m行，每行n个数，描述矩阵a。
第m+1行两个数l、r，表示b矩阵的限制。
输出
第一行一个数为最小的答案。
之后m行每行n个数描述矩阵 b（如果存在多解，输出任意一组即可）。
样例输入
【样例输入一】
2 2
0 1
2 1
0 1
【样例输入二】
2 3
1 2 3
2 6 1
1 3
样例输出
【样例输出一】
1
1 1
1 1
【样例输出二】
2
1 3 1
1 3 3
提示
20%的数据，1 ≤ m, n ≤ 20。
20%的数据，0 ≤ l, r ≤ 1。
100%的数据，1 ≤ m, n ≤ 200，0 ≤ aij ≤ 1 000，0 ≤ l ≤ r ≤ 1 000。
如果你输出的最小值正确，将获得70%的分数，否则将获得0%的分数。
如果你输出的方案也正确，将获得100%的分数。
