1053:GREEN牧场划分grazeset

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
100000kB
描述

农夫约翰的N (4 <= N <= 10,000)头奶牛住在一个圆形牧场中的若干点上。约翰要将这些奶牛分组，所以他想建造K (3 <= K <= 1000)个篱笆，这些篱笆从牧场的圆心开始呈放射状，且相互之间的夹角相等(均为360/K度)。他的目的是合理的安排这些篱笆，使奶牛被分成K个组(当然有些组可能是空的)，定义RANGE为其中最大组中奶牛头数与最小组奶牛头数的差，怎样才能使RANGE值最小？

     当然，没有牛愿意跨骑在篱笆上度日，所以每头牛必须属于某个确定的组。


输入
第1行：N和K。
第2..N+1行：每行有一单个的double型数，表示一头奶牛居住点相对于圆心的角度。 0 <= angle <360。
输出
输出最小的RANGE值
样例输入
4 3
30
60
150.003
240
样例输出
1
