描述

xuxu 和 coco 在玩游戏。

这个游戏包含了 N 个字符串。一开始 xuxu 会从 N 个字符串中选

一个字符串并随便打乱这个字符串内的字符顺序,

然后 xuxu 需要给 出一个 26 个字母的字典序顺序。

xuxu 操作完自己的字符串并且给当出了新的字典序顺序后,

coco 会从剩下来的 N-1 个字符串中选一个并且随意打乱该字符串内的字符顺序。

然后我们将 xuxu 和 coco 打乱后的两个字符串在 xuxu 给出的字典序下进行比较,

谁的字典序小谁就赢。由于 coco 很聪明,

所以 xuxu 一旦选择了不正确的字符串或者

不正确的字典序 coco 就会赢。

xuxu 希望知道,能让自己获胜的选择有 哪些呢??

输入
第一行包含一个正整数 N。
接下来 N 行,每行一个游戏中的字符串。
输出
第一行输出一个整数 K,
代表总共可以让 xuxu 获胜的字符串个数。
接下来 K 行,
每行一个整数代表可以让 xuxu 获胜的字符串的编号。
注意这 K 个整数需要升序排列,且从 0 开始编号。
样例输入
4
a
b
c
d
样例输出
4
0
1
2
3
提示
【数据范围与规定】
对于 100%的数据:n ≤ 50,每个字符串的长度≤50。
