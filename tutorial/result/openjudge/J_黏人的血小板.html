J:黏人的血小板

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
256000kB
描述

在人的体内有许多黏人的血小板，她们彼此依赖，在擦伤中保护血管免受外敌入侵。然而在狭窄的毛细血管，血小板们必须排成一列小心翼翼通过，这可难倒萌萌的血小板们啦，聪明的你快来帮帮她们吧！


输入
第一行为一个整数 n (1 ≤ n ≤ 100000)，为要进入毛细血管的血小板的总数
接下来的 n 行，第 i 行（1 ≤ i ≤ n）包含空格分开的若干个整数，其中第一个整数 mi 表示编号为 i 的血小板所依赖的血小板的个数，接下来的 mi 个整数表示该血小板所依赖的每一个血小板的编号

数据保证单个测试用例中 mi 之和 M ≤ 1000000，全部测试用例中 M 之和不大于 10000000
输出
如果血小板 a 依赖于血小板 b，则 b 必须在 a 前面

如果按照所给的数据，血小板能够排成一列，则输出「Let's go」；否则，输出「Oops」（不包括引号）
样例输入
3
1 2
0
2 1 2
样例输出
Let's go
