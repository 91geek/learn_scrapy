1970:循环汉诺塔

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

想必你一定知道汉诺塔问题,对它的解法肯定也是烂熟于心.现在我们不妨对原来三个柱子的汉诺塔问题做一个扩展,把柱子变成4根,分别标为0,1,2,3,另外,移动规则也略有修改,只允许以下四种移动:从柱子0到柱子1,从柱子1到柱子2,从柱子2到柱子3,从柱子3到柱子0.其他规则不变.初始时所有盘子都在柱子0上,目标是把所有的盘子都移到柱子1上.


请你编写程序解决这样一个循环汉诺塔问题.

输入

输入仅包含一个整数N(1 ≤ N ≤ 15),表示初始时柱子0上的盘子数,盘子的大小都不相同.

输出

输出多行,每行两个整数Ai,Bi表示把柱子Ai最上面的盘子移动到柱子Bi上.移动方法有很多种(不要求一定要最优解),输出任意一种能够达到目标的移动方法即可.


请注意,对任意一组N值,使用的移动步数不得超过100000.

样例输入
3
样例输出
0 1
1 2
0 1
2 3
1 2
0 1
3 0
2 3
0 1
3 0
1 2
0 1
2 3
3 0
0 1
