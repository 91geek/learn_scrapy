0614T3:字符变换

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
65536kB
描述

N个人在玩一个游戏，开始每人手里各有一个长度相同的字符串（仅由大小写字母组成），他们可以对自己的字符串做如下操作：

改变字符串中1个字符，例如abc->abb, abc->Abc

他们被要求对自己的字符串进行M次操作，之后拥有美丽度最高字符串的人获胜。

美丽度定义为：字符串中出现最多的子串（一段不为空的连续字符）的出现次数。

例如：aaaaa = 5， abcabc = 2，ababab = 3

现在如果他们都使用最优策略进行操作，他们能获得的美丽度各是多少？

输入
第一行包含2个正整数N,M
下面N行，每行1个仅由大小写字母组成的字符串，保证长度相同
输出
N行，每位选手能获得的最大美丽度
样例输入
#1
3 3
Kuroo
Shiro
Katie

#2
3 15
foPaErcvJ
mZaxowpbt
mkuOlaHRE
样例输出
#1
5
4
4


#2
9
9
9
提示
* 样例1：Kuroo->ooooo, Shiro->Soooo, Katie->Kaaaa
* 样例2：都能变成 zzzzzzzzz
* 对于40%的数据：1 < N <= 10，字符串长度 <= 10, M = 1
* 对于100%的数据：1 < N <= 100，字符串长度 <= 1000， M <= 1e9
