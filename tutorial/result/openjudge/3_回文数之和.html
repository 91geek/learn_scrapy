3:回文数之和

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

一个正整数被称为回文数当且仅当它的十进制表示从两头读都一样，如131和1441都是回文数，但是123不是。

给出一个正整数N，求所有的(A,B)的个数。使得A+B=N且A与B都是回文数。

例如N=391，那么一共6种方案：

  8 + 383 = 383 + 8 = 391

88 + 303 = 303 + 88 = 391

99 + 292 = 292 + 99 = 391

所以你的程序需要输出6。

输入
输入文件psum.in，一行，包含一个正整数N。
输出
输出文件psum.out，一行，包含方案数。
样例输入
391
样例输出
6
提示
【数据规模】
在30%的数据中，N ≤10^6
在60%的数据中，N ≤10^9
在80%的数据中，N ≤10^14
在100%的数据中，N ≤10^18
