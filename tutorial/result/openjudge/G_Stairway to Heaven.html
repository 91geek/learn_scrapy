G:Stairway to Heaven

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
256000kB
描述

A long time ago in an island far far away, once there was a stairway to heaven. Made of stones, the steps were steep and slip. Starting from the step number one, one can only move one step up or two, but no way back. In a journey that never returns, many fail and die. And the god said, that once every possible path to heaven were tried, there will be finally a survivor. (No survivor before that, and no dead people after that. )


输入
N, the number of steps in the stairway to heaven. 1 ≤ N ≤ 2*10^9.
输出
How many people died in this event before the first survivor. In case the result would be large, output the result modulo 1000000007.
样例输入
2
样例输出
1
提示
Starting from step number one, one can only move one step up to step number two, who's the only people dead in this event.
