描述

zht在打CS,有n个敌人都在运动,每个人有一个出现时间ti,速度vi,和一个身高li(ti<ti+1),zht的狙击枪只有一发子弹,由于他太强了,他可以用这个子弹去打死k个敌人,但是这枚子弹打到的人都必须比上一个人的vi,li都要小
现在zht想要向yjq炫耀他的百步穿杨神技,他打算让这个k最大
同时每个敌人非常担心自己被打死,他们想要你告诉zht打死他们的概率

输入
输入n
然后依次输入vi,li
输出
第一行输出 maxk
接下来依次输出每个敌人被打死的概率(保留5位小数),中间用空格隔开
样例输入
4
1 1
1 0
0 1
0 0
样例输出
3
1.00000 0.50000 0.50000 1.00000
提示
5% n<=10;
10% n<=30;
40% n<=5000;
100% n<=100000;
li,vi<=1e9
