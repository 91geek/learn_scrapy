描述

What is password, password is a word or string of characters used for
user authentication to prove identity or access approval to gain access
to a resource , which is to be kept secret from those not allowed access.

So for programmer, binary expression is the simplest encrypting tool.
But sadly, Mr.xiuqes is so suck at binary stuff and he is forced to
decipher a password, which is written in ASCII code.

Can you help him?

输入
A very long string containing only '0' and '1' represents the ASCII code.
输出
The string the ASCII code represents.
样例输入
011110000110100101110101011100010110010101110011
样例输出
xiuqes
