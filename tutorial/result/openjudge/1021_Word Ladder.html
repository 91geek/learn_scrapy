1021:Word Ladder

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

A word ladder is a connection from one word to another formed by changing one letter at a time with the constraint that at each step the sequence of letters still forms a valid word.

For example, here is a word ladder connecting "code" to "data".
    code -> core -> care -> dare -> date -> data

That word ladder, however, is not the shortest possible one. Although the words may be a little less familiar, the following ladder is one step shorter:
    code -> cade -> cate -> date -> data

Your job in this problem is to write a program that finds a minimal word ladder between two words.
　

输入

The first two lines of the input are two words, the start word and the destination word.
Then n(2 ≤ n ≤ 6000) lines follow, one word per line. It is the lexicon you will use.
It is guaranteed that no reduplicate words in the lexicon and all the words are lowercase.
The input is terminated by EOF. 　

输出

Output contains a single integer, the minimal length of the word ladder. If there is no connection between the two words, output -1.

样例输入
code
data
cade
care
cate
code
core
dare
data
date
样例输出
5
