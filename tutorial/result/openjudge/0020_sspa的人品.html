0020:sspa的人品

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65535kB
描述

sspa最近在玩RPG,人品一直不是很好的他，碰到了一个非常拼人品的地图，地图是这样的，开始sspa在1号传送阵，目标是n号传送阵，在任意一个传送阵有p1%的概率回到上一个传送阵，p2%的概率依然在这个传送阵，p3%的概率进入下一个传送阵,p4%的概率回到1号传送阵。人品非常不好的sspa已经尝试了几次但是都失败了。现在sspa把问题给你了，希望你告诉sspa他开始从1号传送阵到n号传送阵的传送次数的期望是多少？


输入
多组测试数据。
每组测试数据第一行一个整数n，表示有多少传送阵。(n<=1000)
接下来的n-1行，每行3个整数，表示p1,p2,p3,p4。（p1+p2+p3+p4==100）。
输出
每个测试数据一行，表示：从1号传送阵到n号传送阵的传送次数的期望。如果可能到达输出“impossible”(保留2位有效数字)。
样例输入
2
0 100 0 0
2
0 0 100 0
样例输出
impossible
1.00
提示
sspa
