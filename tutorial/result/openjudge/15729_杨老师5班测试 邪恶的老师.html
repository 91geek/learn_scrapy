15729:杨老师5班测试 邪恶的老师

描述

【题目背景】

归功于5班学生的聪明才智和高超的编程技巧，他们拆除了所有的攻击型零件。邪恶的老师感觉很没面子，开始让5班同学罚站，于是有了下面这道题。

【题目描述】

在时长为n秒，学生数为m的一节课里，老师每秒对学生释放魔法，使区间[l,r]的学生站着，半蹲，或者倒立。一开始所有的学生都是站着的，每个学生每站1秒耗费3点体力，半蹲耗费5点体力，倒立耗费10点体力。求一节课所有学生耗费的总体力。


输入
第一行两个整数n，m；
接下来n行，每行三个整数，op，l，r；
如果op=1，表示把[l,r]里的学生都变成站立的状态；
如果op=2，表示把[l,r]里的学生都变成半蹲的状态；
如果op=3，表示把[l,r]里的学生都变成倒立的状态。
输出
一行一个整数，学生耗费的总体力。
样例输入
5 5
2 3 5
1 1 5
3 2 3
3 3 4
2 1 3
样例输出
129
提示
【样例解释】
第一秒，五个学生的状态为1 1 2 2 2，耗费的体力为3+3+5+5+5=21；
第二秒，五个学生的状态为1 1 1 1 1，耗费的体力为3*5=15；
第三秒，五个学生的状态为1 3 3 1 1，耗费的体力为3+10+10+3+3=29；
第四秒，五个学生的状态为1 3 3 3 1，耗费的体力为3+10+10+10+3=36；
第三秒，五个学生的状态为2 2 2 3 1，耗费的体力为5+5+5+10+3=28.
总体力为21+15+29+36+28=129.
【数据范围】
对于60%的数据，n<=1000,m<=1000.
对于其余的40%数据，n<=100000,m<=10000,每次操作的学生数不超过m/10.
