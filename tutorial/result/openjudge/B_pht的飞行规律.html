B:pht的飞行规律

1024000kB
描述

最近，syc在饶有兴致的观察pht的飞行。经过几天的努力，他发现pht的飞行十分有特点：它飞行时，先滑翔一段时间，到达一个最低点后h，再继续上升，整个过程耗时N秒。为了方便记录，syc只测量了pht每个整秒时刻的飞行高度HI (碰巧的是pht在整秒时可到达最低点)(1≤i≤n, i∈N)（至于他怎样测量的就不得而知了）。回家后，syc分析数据惊奇的发现：

       H1 = A  H­­­­N=B

HI = ( HI-1 + HI+1)/2 – k (1 < I < N )  h<= HI

    syc向你提出了一个问题：在何时到达最低点h后上升使HN最大？


输入
文件只有一行：n , A, h, k 。 n， 3 <= n <= 100000，为正整数。 1 < A, k ,h< 109, A, k ,h为实数。
输出
文件只有一行：最大可行的B。(保留二位小数)
样例输入
532.81 0 1
样例输出
446113.34
