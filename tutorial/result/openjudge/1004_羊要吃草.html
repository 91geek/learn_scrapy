1004:羊要吃草

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

现有排成一行的N只羊，从左向右依次编号为1,2, ...,N，其中第k只羊的价值指数记为Ak。

羊群的主人需要分配草给每一只羊，主人认为相邻的两只羊 ，价值指数较大的那只羊应该分配更多的草，但是他又希望所有羊获得的草的总量越少越好。

所以，他希望分配总量尽可能少的草，使得：

（1）对于任意相邻的两只羊，如果两者价值指数不相等，那么，价值较大的羊所得到的草比较小的至少要多M（M>0）个单位草。

（2）每一只羊至少会得到M个单位草。

现在，你的任务是计算草的总量的最小值。

输入
输入第一行包含一个整数T，表示测试数据的组数；
对于每一组测试数据，第一行包含两个整数N和M，表示羊的数量和每只羊会得到的草的最小量；接下来的一行，依次包含N个整数A1, A2, ..., AN，分别表示每只羊的价值指数。
数据范围：
50%：1 <= T <= 20，1 <= N <= 20，1 <= M, A1, A2, ..., AN <= 100；
100%：1 <= T <= 20，1 <= N, M <= 100000，1 <= A1, A2, ..., AN <= 100000。
输出
对于每一组测试数据，输出测试序号（从1开始），以及总量的最小值（草的计量是整数单位，所以答案为整数）。
样例输入
3
1 1
1
2 1
1 2
3 1
1 2 2
样例输出
Case #1: 1
Case #2: 3
Case #3: 4
提示
对于第3组测试数据，分配的草量依次是1,2,1。

共2个测试点，各20分，共40分。
