2012:潘多拉星球的棒球比赛

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
3000ms
内存限制:
65536kB
描述
潘多拉星球的棒球比赛和地球上的有点像。在比赛场地上有四个垒，分别叫做“本垒”、“一垒”、“二垒”、“三垒”。开始，所有的垒上都没有人。参加比赛的两支队伍，一支是进攻队，一支是防守队。

进攻的球队，每次出一个队员走上本垒，去击打防守队投出的球。

每次击球的结果，可能为以下四种之一：

1) “出局”：击球者没打到球，就叫“出局”。在这种情况下，击球者将退出比赛。
2) “上垒”：击球者打到了球，但是球没有飞出比赛场地，就叫做“上垒”。在这种情况下，击球者可以从本垒走到一垒，而他已经在垒上的队友，都可以走到下一垒（三垒的下一垒是本垒）。如果一个进攻队员回到了本垒，他就为进攻队赢得一分。
3) “全垒打”：击球者将球击出比赛场地，就叫“全垒打”。在这种情况下，所有在垒上的进攻队员，包括击球者，都将回到本垒，而且每人都能得一分。显然，如果发生了“全垒打”，进攻队至少得1分，最多得4分。
4) “牺牲打”：击球者放弃击球并退出比赛，叫“牺牲打”。在这种情况下，其他在垒上的队友，可以行进到下一个垒。走回本垒的队员，当然就赢得一分。但是，如果此前“出局”和“牺牲打”的次数之和已经达到了两次，那么，本次“牺牲打”将使比赛立即结束，并且进攻队在本次“牺牲打”中不能得分。

根据规则，进攻队员一旦为本队得分后，就必须立即退出比赛。在所有进攻队员都完成击球后（“牺牲打”也算完成击球），比赛结束。一旦 “出局”和“牺牲打”的次数之和达到了3次，比赛也立即结束。

阿凡达是一支进攻队的教练，他清楚地知道，同一个队员，当他的出场次序不同的时候，表现也将不同。比如，如果你让杰克第一个打，他肯定会“出局”，如果让他第三个打，他就一定会打出“全垒打”。阿凡达非常了解他的队员。他请你为他们的队员安排一个最佳的出场顺序，以便获得最多的分数。
输入
有多组测试数据，以一个0结尾

对每组测试数据:

第一行是一个整数n（1<=n <=15）,表示进攻队队员的数目。所有的队员从1至n进行编号。

接下来是一个n×n 的整数矩阵A，Aij 代表编号为i的队员如果第j个上场，他肯定会打出什么结果(i,j都从1开始算)。

下面是Aij 取值的含义：
0 表示 “出局”
1 表示 “牺牲打”
2 表示 “上垒”
3 表示“全垒打”
输出
对每组测试数据，输出一行，内含阿凡达的球队最多能取得的分数。
样例输入
5
0 2 0 1 2
1 0 2 0 2
0 2 1 2 0
0 2 2 1 2
2 1 0 2 0
0
样例输出
2
