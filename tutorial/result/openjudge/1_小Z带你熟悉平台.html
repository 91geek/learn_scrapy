描述

小Z发现很多人还没有尝试使用这个平台，今天小Z带大家熟悉一下做题的流程。

以一道题目为例，题目描述如下：

在一个长度为n(n<=10)的正整数数列中找出最大值。

很简单吧？就用这题来给大家说明一下平台的各种反馈的含义，请大家继续往下阅读。请大家使用G++测试下面的程序。  

首先是正解，提交这个程序可以获得Accepted的结果。

#include

int main(){

    int T; scanf("%d",&T);

    while(T--){

        int n,a[11],ma=-1;

        scanf("%d",&n);

        for(int i=0; i<n; i++) scanf("%d",&a[i]);

        for(int i=0; i<n; i++) if(ma<a[i]) ma=a[i];

        printf("%d\n",ma);

    }

    return 0;

}

当然，当思路有误的时候就会出现Wrong Answer，如下面这段程序

#include

int main(){

    int T; scanf("%d",&T);

    while(T--){

        int n,a[11],ma=-1;

        scanf("%d",&n);

        for(int i=0; i<n; i++) scanf("%d",&a[i]);

        for(int i=0; i<n-1; i++) if(ma<a[i]) ma=a[i];

        printf("%d\n",ma);

    }

    return 0;

}

Compile Error是最好理解的，编译错误嘛，回去编译器里检查一下吧~

#include

int main(){

    scanf("%d",&T);

    while(T--){

        int n,a[11],ma=-1;

        scanf("%d",&n);

        for(int i=0; i<n; i++) scanf("%d",&a[i]);

        for(int i=0; i<n; i++) if(ma<a[i]) ma=a[i];

        printf("%d\n",ma);

    }

    return 0;

}

常见的错误还有Runtime Error，即运行时错误，当越界、除零等情况时发生，如

#include

int main(){

    int T; scanf("%d",&T);

    while(T--){

        int n,a[11],ma=-1;

        scanf("%d",&n);

        for(int i=-1; i<n-1; i++) scanf("%d",&a[i]);

        for(int i=-1; i<n-1; i++) if(ma<a[i]) ma=a[i];

        printf("%d\n",ma);

    }

    return 0;

}

Time Limit Exceeded的意思是超时，每道题目都有一个时限，必须在规定时间内解出答案才可以，如本题的时限是1s。当然，不是所有题目的时限都这么宽松，大家要小心哟~

int main()

{

    int T = 2;

    while(T++)T--;

    return 0;

}

Memory Limit Exceeded和超时类似，是超过了限定空间。如果发生了，大家要想着给自己的程序“瘦瘦身”哟~

int a[100000001];

int main()

{

    for(int i=0; i<=100000000; i++) a[i]=i;

    return 0;

}

最后要跟大家说一下，下面的样例输入只是给大家一个提示，后台还有很多数据。如果你过了这组样例，并不代表你一定能通过这题哟~比如用下面的代码，就会直接得到Wrong Answer的反馈。

#include

int main()

{

    puts("3");

    puts("5");

    return 0;

}

输入
第一行是一个整数T，代表有T组样例。
接下来T行，每行首先一个整数n(1<=n<=10)，代表数列长度。紧接着n个整数，代表这一组正整数数列。
输出
对于每组数列，输出一行，包含一个数字，代表最大值。
样例输入
2
3 1 2 3
5 1 2 3 4 5
样例输出
3
5
