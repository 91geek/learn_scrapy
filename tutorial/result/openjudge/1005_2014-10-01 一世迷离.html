20000ms

262144kB
描述

yzy的姐姐叫yyy。他们有一块大小为n*m的矩形的祖传的土地。这块土地被平均分成
n行m列，总共n*m格。有一些格子上有树，其它的格子上则没有。
现在他们打算找出一块正方形的土地来种植仙人掌（正方形的边界要与格线重合），当
然这块土地上不能有树。可是他们最多只能砍掉k个格子里的树。yyy想知道他们最大能找
到多大面积的土地。

输入
第一行三个正整数n, m, k，表示土地的行数，列数和最多能砍掉的树的数目。
接下来n行，每行一个长度为m的仅由'T'和'.'构成的字符串，表示这块土地。如果
第i行第j列的字符为'T'，则这格土地上有树，否则这格土地是平坦的。
输出
一行一个正整数，表示最大的土地面积。
样例输入
3 4 2
..T.
.T..
T...
样例输出
9
提示
对于50%的数据，1 ≤ n, m ≤ 50。
对于80%的数据，1 ≤ n, m ≤ 300, 0 ≤ k ≤ n * m。
对于100%的数据，1 ≤ n, m ≤ 1000, 0 ≤ k ≤ n * m。
