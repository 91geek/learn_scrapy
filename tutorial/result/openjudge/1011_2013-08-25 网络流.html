描述

xuxu 和 coco 在前线。

在前线有 N 个士兵,每个士兵有生命值、攻击力和防御力,

且这 三种属性的数值均为小于等于 K 的正整数。

如果对于第 i 名士兵,有 k 名士兵的生命值、 攻击力、

防御力均小于等于第 i 名士兵的生命值、 攻击力、防御力,

那么第 i 名士兵的等级就为 k。

给出每个士兵的生 命值、攻击力、防御力,问每个等级的士兵有多少个。

输入
第一行两个整数,代表 N、K。
接下来 N 行,每行三个整数,代表士兵的生命值、攻击力和防御
力。
输出
总共 N 行,每行一个整数,代表每个等级的士兵数量。
样例输入
10 3
3 3 3
2 3 3
2 3 1
3 1 1
3 1 2
1 3 1
1 1 2
1 2 2
1 3 2
1 2 1
样例输出
3
1
3
0
1
0
1
0
0
1
提示
【数据范围与规定】
对于20%的数据:1≤N≤100,1≤K≤100。
对于另外20%的数据:1≤N≤100,000,1≤K≤100。
对于另外20%数据:1≤N≤100,000,1≤K≤1,000。
对于另外20%数据:1≤N≤50,000,1≤K≤100,000。
对于100%的数据:1≤N≤200,000,1≤K≤200,000。
