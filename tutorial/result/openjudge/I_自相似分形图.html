I:自相似分形图

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

考虑一个三角形。如果我们把它按三边中点分成四个全等的小三角形，去掉中间的那个，再对剩下的三个作相同的分割操作，这样如此下去就会得到面积为0的“三角形”区域。这个分形图被叫做谢宾斯基三角形。尽管其拓扑维度为2，但其豪斯多夫维约是1.58(log(3)/log(2))，并不是整数（这也就是“分形”这个名字的来历）。

在这个问题中，对一个三角形，要求你仅使用ASCII字符画出它经过一定次数的分割所形成的谢宾斯基三角形。由于基本单元的大小固定，图形的大小应当随着递归层数的增长而变大。基本图形如下，用两个斜线(/)，两个反斜线(\)和两个下划线(_)画出：

 /\
/__\

欲知如何画出更大的三角形，请看样例输出。

输入
一个整数n，0 <= n <= 9，为分割次数。
输出
用ASCII字符画出一个基本单元经过n次分割之后形成的谢宾斯基三角形。注意，你的输出的最左下角的斜线(/)左边不应当有空格。
样例输入
Test case 1:
2
Test case 2:
1
Test case 3:
0
样例输出
Test case 1:
       /\
      /__\
     /\  /\
    /__\/__\
   /\      /\
  /__\    /__\
 /\  /\  /\  /\
/__\/__\/__\/__\
Test case 2:
   /\
  /__\
 /\  /\
/__\/__\
Test case 3:
 /\
/__\
提示
分割次数为6的谢宾斯基三角形。
