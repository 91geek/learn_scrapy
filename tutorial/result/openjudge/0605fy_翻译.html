天平

描述

Iyo Kiffa-Australis 女士有一个天平和一套只有两种重量的砝码，用以称量药品的质量。例如，使用300 毫克和700 毫克砝码去称量200 毫克的阿司匹林，可以在200 毫克药品那一边放置一个700 毫克砝码，另一边用三个300 毫克砝码（图一）。尽管另一种方法，即在200 毫克药品那一边放置四个300 毫克砝码，另一边用两个700 毫克砝码（图二）也行，但因为偷懒就不会这样解决。

现在你被邀请去帮她计算需要砝码的个数。


输入

输入包含若干份数据序列。一份数据在一行中输入三个用空格隔开的正整数a、b 和d。 满足条件：a != b，a <= 10000，b <= 10000 并且d <= 50000。你可以先假设用a 毫克和b 毫克砝码的组合，能够称量出d 毫克。换句话说，你应当考虑「无解」的情况。

输入以一行连续的三个0 结束。这不是分数据。

输出

输出包含与输入对应的若干份数据（a，b，d）。一行输出包含两个以空格隔开的非负整数x 和y，并满足以下条件。

你可以用x 个a 毫克和y 个b 毫克砝码称量出d 毫克；

数量的总和（x + y）在保证以上条件时，为最小值；

质量的总和（ax + by）在保证以上两点条件时，为最小值；

严格控制输入，将不接受多余的字符（如多余的空格）。

样例输入

700 300 200
500 200 300
500 200 500
275 110 330
275 110 385
648 375 4002
3 1 10000
0 0 0

样例输出

1 3
1 1
1 0
0 3
1 1
49 74
3333 1

来源

Japan 2004

















C 语言的循循循环

描述

一个迷之编译问题：我们有一个C 语言风格的循环类型：

for (variable = A; variable != B; variable += C)
        statement;

换句话说，一个循环的定义是——首先设置一个变量A，只要当它与B 不相等时，就重复执行某一语句，并增加变量C。我们想知道在A、B 和C 的调控下，语句执行的次数。假设所有的变量及计算结果都是一个k 位的无符号整型，即对2k求模（即数值x 满足，0 <= x< 2k）。

输入

输入包含若干情况。每一种情况都输入为一行，包括四个被空格隔开的整数A、B、C 和k。整数k（1 <= k <= 32）意为控制循环及其参数变量A、B、C（0 <= A, B, C < 2k）的位数。

输入以一行连续的四个0 结束。

输出

输出包含与输入对应的若干行。第i 行输出循环的次数（为一个单行整型数字）。若循环不会终止，则输出「FOREVER」。

样例输入

3 3 2 16

3 7 2 16

7 3 2 16

3 4 2 16

0 0 0 0

样例输出

0
2
32766
FOREVER

来源

CTU Open 2004
