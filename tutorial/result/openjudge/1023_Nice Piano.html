描述

Fungi has recently bought himself a nice 88-key piano! A piano has one octave with 7 white keys and 5 black keys that look like this


=============================

|  [#] [#]  |  [#] [#] [#]  |

|  [C] [D]  |  [F] [G] [A]  |

|  [4] [4]  |  [4] [4] [4]  |

|  [_] [_]  |  [_] [_] [_]  |

|   |   |   |   |   |   |   |

| C | D | E | F | G | A | B |

| 4 | 4 | 4 | 4 | 4 | 4 | 4 |

=============================


The white keys are named as shown. However, the keys may also be named from its adjacent keys. Typically the black keys between white keys are named either '#' (sharp) plus its left side key, or 'b' (flat) plus its right side key. For example, the first black key in the above picture would be named #C or bD. Note that only one sharp or flat symbol is allowed at a time.


The piano has seven full octaves numbered from 1 to 7. To differentiate the keys with same names in different octaves, the octave number are tailed at the key name, e.g, the middle C in octave 4 is named C4. However, there are 4 additional keys: A0, #A0, B0 at the leftmost, and C8 at the right most, making a total number of 88 keys.


To practice his drawing skill, Fungi wants to draw his lovely piano.

输入
The first line has an integer T, the number of test cases.
Each test case has one integer, k (k<=88), specifying the number of keys to draw. Following that, a key name is given. The key name is guaranteed to be valid on the piano.
输出
For each case, draw k keys from left to right starting from the given key. However, do not draw out of the piano's key range. If the leftmost or rightmost key has an adjacent black key to its left or right, draw half of the black key contour only. When drawing key names, use no sharp or flat symbols for white keys, and sharp symbol only for black keys. Refer to the sample output. Pay attention to the white spaces!
样例输入
3
12 C4
6 bC6
4 #F2
样例输出
=============================
|  [#] [#]  |  [#] [#] [#]  |
|  [C] [D]  |  [F] [G] [A]  |
|  [4] [4]  |  [4] [4] [4]  |
|  [_] [_]  |  [_] [_] [_]  |
|   |   |   |   |   |   |   |
| C | D | E | F | G | A | B |
| 4 | 4 | 4 | 4 | 4 | 4 | 4 |
=============================
