描述

The objective of the Number Placement Puzzle is to fill a grid with integers from to so that the following conditions hold:

1. All integers on the main diagonal, which runs from top left to bottom right, are zero.  
2. All integers except those on the main diagonal are positive.
3. If there is an integer X at the cross grid of the -th row and the -th column ( is not equal to ), integer X does not appear in the -th row or the -th column.

Sometimes is too small to solve the puzzle. Given , please find a solution with minimized . Since there might be multiple solutions, you may output any one of them.

Please refer to the sample output for better understanding.

输入

The first line contains an integer, indicating the number of test cases, which is no more than .

Each test case consists of a single line, and each line contains an integer , indicating the size of the grid.

输出

For each test case:

If no such solution exists, output the emoticon "QaQ" (without quotations) in a single line. Otherwise, output lines, each line contains integers separated by spaces. The -th integer in the -th line indicates the integer you assigned to the cross grid of the -th row and the -th column.

5
2
