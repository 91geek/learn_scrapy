08:Sorting

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

Given an integer sequence, we define a new operation: select a group of consecutive(连续的) elements of within the integer sequence and move them to another place of the sequence with the original order. For example, for the following sequence “5 4 1 2 3”, we can choose “1 2 3” and then insert them into the first place to form a new sequence “1 2 3 5 4”. We want to know how many steps we need at least to make the sequence in the ascending order (“1 2 3 4 5” in this example).


输入
The first line is an integer T (T<=10), which indicates the number of test cases.
For each test case, there are two lines. The first line is an integer N (N<=15), which is the length of sequence. The second line contains N integers representing the sequence.
输出
For each test case, the output is an integer, which is the minimal number of operations. If it is larger than 5, please print “Can not finish in 5 steps”
样例输入
3
5
2 1 4 3 0 
12
5 0 9 1 3 2 10 11 6 4 8 7 
9
9 8 7 6 5 4 3 2 1
样例输出
Case #1: 2
Case #2: 5
Case #3: Can not finish in 5 steps
