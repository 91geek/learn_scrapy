2:Bg的报复行动

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述
wft和ql回来了，bg也和他们一起回来了。bg对于上次被耍感到非常生气，这次他打算报仇。这个星期天，wft和ql在玩躲猫猫，bg意识到他的机会来了。此刻，wft正在地点A闭着眼睛数数，ql躲在与A地点有一定距离的B地点。bg站在地点S,现在bg想要将他们个一个推倒，同时立刻逃跑到T地点。然而问题来了，由于刚下完雪，地上非常滑。bg想快速做完上述事情，同时在他路线上任何一点的曲率半径不能大于R。为了吓到他们，bg决定从他们的背后推倒他们，也就是说在bg经过要推的人时，他必须和那个人面向同一个方向。请帮助bg策划一条满足上述条件的最短路。为了简化问题，你只须求出路径的长度
输入
输入包含多组测试数据，每组数据一行，以EOF结束。对于每组测试数据，第一行包含一个整数R，接下来的四行描述地点S,A,B,T的坐标。
XS, YS
XA, YA, DXA, DYA
XB, YB, DXB, DYB
XT, YT
DX和DY表示wft和ql所面对的方向。所有的数据的绝对值不会超过100.你可以假设bg推倒别人的时候不会改变自己的方向。
输出
对于每组数据输出一行，bg行动的最短路径长度，保留两位小数。
样例输入
1
0 0
1 0 1 0
2 0 1 0
3 0

1
0 0
1 0 -1 0
2 0 -1 0
3 0

1
0 0
1 0 1 0
1 3 -1 0
0 3

1
0 0
1 1 0 1
0 2 -1 0
-1 1
样例输出
3.00
13.14
6.14
4.71
