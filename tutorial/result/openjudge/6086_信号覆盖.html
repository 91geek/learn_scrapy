6086:信号覆盖

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

为了探索海洋中的奥秘，某国家的海洋勘探中心在太平洋中安装了n个数据采集器。为了控制这些数据采集器，他们在海洋中修建了三个信号发射器从而实现了无线遥控。可这套系统已经用了很长时间了，信号发射器已经老化，需要更换。恰巧此时正值金融危机，国家财政非常紧张，于是海洋勘探中心决定使用尽量便宜的信号发射器。技术人员表示，信号发射器的价钱等于它工作半径的平方乘以它的费用参数，并且每个数据采集器只要被任意一个信号发射器控制（即在这个信号发射器的工作范围内）就能正常工作。


现在就让你帮忙确定每个信号发射器的工作半径，在保证所有数据采集器可以正常工作的情况下，费用最小。

输入

前三行中，每行有4个非负整数Xi、Yi、Zi、Pi表示每个信号发射器的位置和费用参数。


第四行为1个整数n，表示星球个数。


后面n行每行3个非负整数xi、yi、zi，表示每个数据采集器的位置。

输出

一个整数，表示最小的费用。

样例输入
5 0 0 1
0 5 0 2
0 0 5 1
6
10 0 0
0 9 0
0 0 8
5 5 0
0 4 4
1 1 1
样例输出
68
