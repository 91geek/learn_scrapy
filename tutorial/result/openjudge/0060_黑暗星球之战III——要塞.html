0060:黑暗星球之战III——要塞

30000ms
单个测试点时间限制:
3000ms
内存限制:
256000kB
描述

“En Taro Light! 反制器被成功破坏，未能按照计划工作，现在我们可以直接攻击A0A0的前线基地——黑暗要塞了。”——Light联盟前线指挥官
　A0A0的反制器没能发挥期望的作用，导致了A0A0的要塞完全暴露在了高能射线的火力范围内，但幸运的是，由于光明射线的能量也不算太高，射线并不一定能完全摧毁要塞。现在前线指挥官想对这次打击的效果进行一个预估，跟往常一样的，这个任务又交给了你。　

A0A0的要塞由K个房间组成，由K-1个通道将其连接，对于每个房间有Pi%的概率被直接摧毁，对于每个通道有Di%的概率被击穿。假如一个房间被直接摧毁，这个房间通过被击穿的通道可到达的所有房间也会被一并摧毁。为了评估这次的打击质量，请你算出期望意义下摧毁房间的个数。

输入
第一行一个整数K，表示房间数。
后接K-1行，每行3个整数x,y,di，表示一条连接x和y的击穿概率为di%的通道。
最后一行K个整数，第i个整数Pi表示i房间有Pi%概率被直接摧毁。
输出
输出一个实数，表示期望下摧毁房间的数量（保留6位小数）。
样例输入
3
1 2 50
1 3 50
50 0 0
样例输出
2.000
提示
【数据范围】
　　对于30%的数据，K <= 50。
　　对于100%的数据，K<=500000。
