C:全知全能的神

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
256000kB
描述

众所周知，孙神是 nil 星的唯一真神。一个人管理整个星球对孙神来说是很轻松的，所以他喜欢在闲暇之余打台球。这一天他创造了一个有 n 个顶点的多边形台球桌。他想要给台球桌的每个角落配备球洞和球。每个球必须能且只能与其中的一个球袋配对。因为孙神击球会向球中注入神之力，所以球在运行与反弹过程中，无动能损耗。

要求：1. 每个顶点只能有一个球或球袋；

    2. 击球时只能沿45°角击球；

    3. 球必须能进入它所配对的球袋。

问最多能设置多少球袋。


输入
第一行包含一个整数N（4<=N<=100000)表示一共N个顶点
接下来N行，每行两个整数Xi,Yi(-100000<=Xi, Yi<=100000)表示每个顶点的坐标（题目保证给的点按顺时针顺序）
输出
一个整数，表示最多能设置多少球袋
样例输入
10
1 1
3 1
3 -2
-3 -2
-3 0
-1 0
-1 -1
2 -1
2 0
1 0
样例输出
5
提示
分别设置为(-3,0),(-1,0),(-1,-1),(2,-1)(2,0)
