描述

小A喜欢Fibonacci数，小A也喜欢上了Fibonacci串，它们是这样定义的：

•Fib1 = b

•Fib2 = a

•FibN+2 = FibN+1 + FibN, N ≥1

在这里加法是字符串的连接。小X还很喜欢一单词，他想知道这个单词在某个Fibonacci 串中出现的次数。


输入
第一行包含一个长为 M 的仅有 a 和 b 组成的字符串，表示单词。
第二行包含一个正整数 N，表示是第 N 个 Fibonacci 串。
输出
仅一行包含一个非负整数，表示答案。
样例输入
aba
6
样例输出
3
提示
对于 100% 的数据，1 ≤ M ≤ 30, 1 ≤ N ≤ 200。
数据有梯度。
