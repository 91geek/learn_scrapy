1059:四散的玩具

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

longo从黑市上新买了一盒玩具，回到家以后迫不及待的将所有的玩具摊到了地面上。这时他惊奇的发现这些玩具居然都是一些三角形的卡片，并且在三个角上分别印有红，绿，兰（之后用R，G，B表示）三种颜色之一（同一种颜色可能同时出现在一个卡片上多次）。而在这之中，其实有好多张卡片属于同一种。当且仅当两张卡片经过旋转，翻折以后能够完全重合，且对应角的颜色完全相同，我们认为这两张卡片是属于同一种类的。

地面可以看做是一个平面直角坐标系，现在每个三角形卡片的三个顶点坐标与其对应的颜色都已知。当然由于是胡乱撒了一地，卡片之间相互是可能重叠的。

由于着急的longo想要按照类别整理一下这些玩具，于是可以请你帮他数数看这盒玩具中总共有多少种不同类型的卡片么？

输入
第一行一个整数T，代表数据组数。
对于每组数据，第一行有一个整数n(1<=n<=1000)，代表地上的卡片总张数。
接下来的n行，每行有三对形如(x,y,C)的三元组，分别代表一个三角形卡片的三个顶点在地面上的坐标(x,y)以及这个顶点的颜色C。其中x,y为整数，且-10000<=x,y<=10000，C∈{'R','G','B'}。
输出
对于每组数据，输出一个整数，为该盒玩具中所拥有的卡片总种类数。
样例输入
2
2
-1 1 R 0 -1 G 1 0 B
0 1 G -1 0 B 1 -1 R
3
0 0 R 0 1 R 1 0 B
0 0 R 0 1 B 1 0 B
-1 0 R 0 1 R 1 0 B
样例输出
1
3
