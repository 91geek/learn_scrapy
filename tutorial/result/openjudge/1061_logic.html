描述

lcch迎来了新的学期，在新的学期中lcch学习到了一门神奇的课程————数理逻辑。lcch被里面的a蕴含b，非a蕴含b，a蕴含非b,非a蕴含非b弄的心情非常烦躁。

（a蕴含b，表示如果a为真则b必为真。非a蕴含b，表示如果a为假则b必为真。a蕴含非b，表示如果a为真则b为假。非a蕴含非b表示如果a为假则b为假）

有一天plato于是想给lcch指导一下，plato给了lcch若干了命题，每个命题都是都是形如a蕴含b，非a蕴含b，a蕴含非b,非a蕴含非b的形式，其中a和b是命题变元。plato告诉lcch这些命题全部都是真命题，他想让lcch推理出哪些命题变元的取值必须为假。

lcch接过plato的题目：“a蕴含c，a蕴含非b, 非b蕴含非c”。lcch思考到：“如果a取值为真，那么c也为真，并且b为假，而b为假可以推出c为假，那么c又真又假。嘿嘿，我知道了，看来a取值必须为假。b和c似乎也可以为真。“lcch解决了plato的题目之后，心想如果命题变元变多了怎么办？这就是你需要解决的问题。


输入
多组数据，以EOF结束。
每组数据第一行两个数n(1<=n<=1000)，m(1<=m<=100000),其中n表示命题变元的个数, m表示命题的个数。
接下来m行，每行4个数表示x, y, ox, oy。(1<=x, y<=n, 0<=ox, oy<=1).
如果ox=1,oy=1，表示x蕴含y
如果ox=1,oy=0，表示x蕴含非y
如果ox=0,oy=1，表示非x蕴含y
如果ox=0,oy=0，表示非x蕴含非y
输出
对于每组数据输出两行，第一行表示取值必须位假的命题变元的个数。接下来一行按照递增的顺序输出必须为假的命题变元的编号。
样例输入
3 3
1 3 1 1
1 2 1 0
2 3 0 0
3 3
1 3 0 0
1 3 1 0
3 2 0 0
样例输出
1
1
2
2 3
