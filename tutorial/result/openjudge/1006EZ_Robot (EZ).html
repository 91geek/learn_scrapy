1006EZ:Robot (EZ)

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

Renne controls a robot on a 2D plane by a command string. Initially the robot is at (x0,y0). The x axis points at south, while the y axis points at east.


There are four commands:

N: move north, x = x-1

S: move south, x = x+1

W: move west, y = y-1

E: move east, y = y+1


Determine the final position of the robot after all the commands are executed.


输入
The input has several cases. Each case has two integers on its first line, x0, y0 (-100<=x0,y0<=100). The second line has a command string no longer than 100.
The input is terminated by "END".
输出
For each case, output the final position of the robot after all commands are executed.
样例输入
0 0
NSWE
10 10
EESSNWWSSEENNNN
END
样例输出
0 0
9 12
