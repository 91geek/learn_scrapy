3001:站军姿

描述

站军姿是一个非常痛苦的事情，特别当教官在附近游荡时

有两个教官，每个教官有一个位置坐标和一个视线范围，该教官能够观察到以该位置为圆心， 给定半径的一个圆，你需要求出有多大面积的地方在两个教官的视线范围内 由于教官在四处游荡，因此你需要实现多组数据

输入
第一行一个数T表示数据组数
接下来T行每行六个实数，分别为 xi,yi,ri,x2,y2,r2, 代表两个教官的各项参数
输出
每组数据输出一行表示答案，答案保留三位小数
样例输入
1
3 3 1 3 3 0.5
样例输出
3.142
提示
对于10%数据，两圆相离
对于10%数据，两圆为包含关系
对于另外40%数据，满足T = 1且0 <= |x|，|y|,r <= 5
对于 100% 数据，T <= 10, 0 <= |x|，|y| <= le5
