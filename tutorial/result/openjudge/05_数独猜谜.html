05:数独猜谜

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
102400kB
描述

一段时间后，HeHe看到他的好友FY在玩数独，于是他也加入了这个行列。可惜的是，HeHe玩了很久水平依然很烂，但是他却挑战大师级的FY，于是他再次找到了你。。希望能帮他立刻填好一个数独


输入
9×9的矩阵。‘0’是需要填的，不能修改非0格
输出
　9×9的矩阵，即填好后的数独。保证每一个数据都有且仅有唯一答案
样例输入
0 3 0 0 0 7 0 0 4
　6 0 2 0 4 1 0 0 0
　0 5 0 0 3 0 9 6 7
　0 4 0 0 0 3 0 0 6
　0 8 7 0 0 0 3 5 0
　9 0 0 7 0 0 0 2 0
　7 1 8 0 2 0 0 4 0
　0 0 0 1 6 0 8 0 9
　4 0 0 5 0 0 0 3 0
样例输出
　8 3 9 6 5 7 2 1 4
　6 7 2 9 4 1 5 8 3
　1 5 4 8 3 2 9 6 7
　5 4 1 2 8 3 7 9 6
　2 8 7 4 9 6 3 5 1
　9 6 3 7 1 5 4 2 8
　7 1 8 3 2 9 6 4 5
　3 2 5 1 6 4 8 7 9
　4 9 6 5 7 8 1 3 2
提示
　非0格数小于等于64
　
提示：数独的规则：满足每一行、每一列、每一宫内的数字均含1-9，不重复。
