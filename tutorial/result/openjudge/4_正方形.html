4:正方形

20000ms
单个测试点时间限制:
2000ms
内存限制:
262144kB
描述

给定直角坐标系中的若干整点，请寻找可以由这些点组成的正方形，并统计它们的个数。

输入
包括多组数据，每组数据的第一行是整点的个数n(1<=n<=1000)，其后n行每行由两个整数组成，表示一个点的x、y坐标。输入保证一组数据中不会出现相同的点，且坐标的绝对值小于等于20000。输入以一组n=0的数据结尾。
输出
对于每组输入数据，输出一个数，表示这组数据中的点可以组成的正方形的数量。
样例输入
4
1 0
0 1
1 1
0 0
9
0 0
1 0
2 0
0 2
1 2
2 2
0 1
1 1
2 1
4
-2 5
3 7
0 0
5 2
0
样例输出
1
6
1
提示
40%的数据，坐标绝对值小于等于1000
