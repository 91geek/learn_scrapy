9:打印变量

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

C语言中存储数据都需要使用叫变量的东西。变量的名字是你可以随便取的，不过一般我们都会起个看得懂的名字，这个名字是唯一的，不能与别的名字重复。变量分为几种不同的数据类型：

常用的一种是专门用来装整数的用 int 表示，

一种是专门用来装字符的用 char 表示，

还有一种是专门用来装小数的用 float 表示。

变量在使用之前都要先声明（定义）后使用，在变量定义的时候可以给其赋值（这个就叫变量的初始化）。

//符号后面是程序的注解，是帮助我们理解的，不影响程序运行。

int applenumber = 10; //定义一个叫applenumber的变量，初始化数据为10

char cha = 'a'; //字符使用的时候必须用单引号引起来

float money = 123.23; //这个是用来装小数的

double r = 1234.123456; //这个也是用来装小数的，只不过精度比较高


如果我们想把上面的这几个变量打印出来该怎么办呢？使用 printf( )，没有错，但是打印上面这些变量会有一些小小不同的地方哦，那么请你在代码中把上面的4个变量分别都打印出来吧！看看你能不能写对呢？

输入
无
输出
printf("%d\n", applenumber); //输出语句中使用 %d 代表整型 \n表示换行
printf("%c\n", cha); //输出语句中使用 %c 代表字符型
printf("%f\n", money); //输出语句中使用 %f 代表小数类型
printf("%f\n", r); //输出语句中使用 %f 代表小数类型
样例输入
无
样例输出
10
a
123.230003
1234.123456
提示
一定要记住哦，打印不同类型的变量使用的字符是不一样的。
