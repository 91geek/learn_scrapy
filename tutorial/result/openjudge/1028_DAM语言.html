1028:DAM语言

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
256000kB
描述
有个机器执行一种DAM语言。该机器有一个栈，初始时栈里只有一个元素x，随着DAM语言程序的执行，栈里的元素会发生变化。DAM语言有三种指令：
D指令：把栈顶元素复制一次加到栈顶
A指令：把栈顶元素取出，加到新的栈顶元素中。注意：此时若两相加元素指数不同，相加后所得结果的指数为两元素指数的最大值，系数仍然相加。例如原栈顶元素为x^3，新的栈顶元素为2x，则相加结果为3x^3。
M指令：把栈顶元素取出，乘到新的栈顶元素中。
如果执行A或M指令的时候栈内只有一个元素，则机器会发出错误信息。如果程序无误，那么执行完毕以后，栈顶元素应该是x的单项式，例如，程序DADDMA的执行情况为（栈内元素按照从底到顶的顺序从左至右排列，用逗号隔开）：
x → x， x → 2x → 2x，2x → 2x，2x，2x → 2x，4x^2 → 6x^2
给出一段DAM程序，求出执行完毕后栈中元素。
输入
仅一行，包含一个不空的字符串s，长度不超过12，代表一段DAM程序。
输出
若执行指令时程序出错，只需输出错误信息’FAILED’；否则输出执行完毕后栈中各元素，相邻元素间用空格隔开。每个元素为一个单项式，须按照习惯写法输出，即：等于1的系数不要打印，一次方不打印 ’^1’。
样例输入
DDDADMADM
样例输出
x 25x^4
