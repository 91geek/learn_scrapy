2048:快速排序

  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述
排序是我们学习编程需要掌握的一个最基本的操作。
这个世界上有很多种排序的方法，比如插入排序、选择排序、冒泡排序，但是他们都不够快。后来聪明的C. A. R. Hoare想出了一种很快的排序方法，由于它的确很快，于是我们就叫它快速排序。
快速排序的原理其实很简单。假设我们需要将一些待排序的数从小到大排序。首先，我们从这些数中随便找一个数（key），然后把比它小的放到它的左边，比它大的放到它的右边，这样它左边的任意一个数都比右边的任意一个数小，我们把这个操作叫做“划分”。
这时候我们再递归调用快排函数，将key的左边、右边都排好，整个集合的数字就有序了。
举个例子：
待排序的数字： { 49 38 65 97 76 13 27 49 }
选择最左边的49为key（这个是随意选择的），进行划分操作。
变成了： { 27 38 13 } 49 { 76 97 65 49 }
然后递归调用快排函数，将key49的左边、右边排序：
左边选择27进行划分：
｛13｝ 27 ｛38｝
右边选择76进行划分：
｛49 65｝ 76 ｛97｝
最后再排一下｛49 65｝
这样就排好了： { 13 27 38 49 49 65 76 97 }

现在，我们要写一个快速排序的函数。如果你还没有看懂快排是什么的话，不用担心，因为整个排序操作的其他部分都已经给出了，我们只需要补写一个划分函数Partition()就好了。

程序的主体是这样的：
#include<stdio.h>
#define N 1000 + 10
int Partition(int a[], int low, int high)；

void Qsort(int a[], int low, int high){
int pivotloc;
if(low >= high)
return;
pivotloc = Partition(a, low, high);//划分操作
Qsort(a, low, pivotloc-1);//对左侧递归排序
Qsort(a, pivotloc+1, high);//对右侧递归排序
}

int main(){
int a[N], i, n;
//输入
scanf("%d", &n);
for(i=0;i<n;i++)
scanf("%d", &a[i]);

Qsort(a, 0, n-1);
//输出
for(i=0;i<n;i++)
printf("%d ", a[i]);
printf("\n");

return 0;
}



现在我们需要你需要编写Partition()这个函数，也就是划分操作。
这个函数要求我们从a数组里的low到high这个范围内随便找一个数字key，把小于key的放到左边，大于key的放到右边，然后返回返回key的位置，不在low到high范围内的数不考虑。
提交的时候请复制上述代码，然后加上自己写的函数。
输入
第一行为一个数字n，0<=n<1000。
第二行包括n个数字，每个数字都在int型范围以内。
只有一组测试用例。
输出
按照从小到大的顺序将这些数字输出。
样例输入
5
2 2 1 0 4
样例输出
0 1 2 2 4
