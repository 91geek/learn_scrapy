2013:传话

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述
多好的天！郊游烧烤去吧！今天，向阳幼儿园的所有孩子都准备好了要去郊游。出发之前，刘老师让大家都站成一排。刘老师有一个重要的消息要宣布，但是她不想直接告诉大家。她想让这个消息以口口相传的形式散布开来。孩子们在把听到的话再告诉别人的时候，很可能就会走样，刘老师想看看最后这个消息会变出多少个版本。所有的孩子身高都不同，刘老师就依据身高规定了下面这些传消息的规则:
1) 她只把消息告诉个头最高的孩子。
2) 每个得到消息的孩子，都必须转告他（她）的“左信使”和“右信史”。
3) 每个孩子的“左信使”，就是这个孩子的“左小弟”中个头最高的那个。
4) 每个孩子的“左小弟”，就是在这个孩子左边，能被这个孩子看见的比他矮的所有孩子。.
5) 当一个孩子向左看的时候，他的目力所及，止于离他最近的比他高的孩子。
6) “右信史”，的定义，和“左信史”类似，只是将“左”换成了“右”。

例如，假设一行孩子的高度从左到右分别为4, 1, 6, 3, 5, 2 在这种情况下，刘老师把消息告诉第3个孩子；然后第3个孩子传话给他的“左信史”---第1个孩子以及他的“右信史”----第5个孩子；接下来第1个孩子传话给第2个孩子，第5个孩子传话给第4个孩子和第6个孩子。

你的任务就是要找出消息传递的路径。
输入
第一行是一个整数T，表示测试数据的组数。接下来是T组测试数据。
每组测试数据有两行。第一行是一个整数N (0< N <= 50000)，表示孩子的数目。第二行从左到右列出了所有孩子的身高。每个孩子的身高都是整数，都不相同，而且小于231 – 1
输出
对于每组测试数据，先输出一行“Case t:” ( t 是测试数据的组编号，从1开始算)。然后再输出N行，其中第i行包含两个整数，表示第i个孩子的“左信使”和“右信使”的编号（i从1开始算，编号也是从左至右从1开始算）。如果一个孩子没有“左信使”或“右信使”那么，输出“左信使”或“右信使”编号的时候就输出“0”。
样例输入
2
5
5 2 4 3 1
5
2 1 4 3 5
样例输出
Case 1:
0 3
0 0
2 4
0 5
0 0
Case 2:
0 2
0 0
1 4
0 0
3 0
