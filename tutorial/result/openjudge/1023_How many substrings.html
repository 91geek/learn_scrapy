1023:How many substrings

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

Given a string, we need to findthe total number of its distinct substrings. We say a string S(i…j) is asubstring of S, where 0 <= i <= j < Length(S), if the index of firstcharacter of string S starts from 0.


输入
The first line contains an integer T, no more than 20, indicating the number of test cases.
Each of the following T lines contains a string which length is no more than 50,000, represents a test case.
Characters are from ‘A’ to ‘Z’ or ‘a’ to ‘z’.
输出
For each case, print the case number beginning with 1 and the number of distinct substrings.
样例输入
2
CCCCC
ABABA
样例输出
Case #1: 5
Case #2: 9
提示
For the 1st case, we have 5 different substrings as follows:
C, CC, CCC, CCCC, CCCCC.
For the 2nd case, we have 9 different substrings as follows:
A, AB, ABA, ABAB, ABABA, B, BA, BAB, BABA.
