描述

写一个函数 void getSumDif(int x, int y, int *pSum, int *pDif);

函数功能是求参数x和y的和与差。

#include <stdio.h>

void getSumDif(int x, int y, int *pSum, int *pDif);

int main()
{
    int x, y, sum, dif;
    scanf("%d%d", &x, &y);
    getSumDif(x, y, &sum, &dif);
    printf("The sum of %d and %d is %d.\n", x, y, sum);
    printf("The difference of %d and %d is %d.\n", x, y, dif);
    return 0;
}
// 在此处补充你的代码
		
输入

		
2个整数

		
输出

		
这2个整数的和与差

		
样例输入

		
8 5

		
样例输出

		
The sum of 8 and 5 is 13.
The difference of 8 and 5 is 3.
