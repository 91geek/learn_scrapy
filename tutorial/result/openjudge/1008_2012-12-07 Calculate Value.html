1008:2012-12-07 Calculate Value

  • 查看
  • 提交

总时间限制:
10000ms
单个测试点时间限制:
1000ms

2560000kB
描述

HYC在LZT的帮助下赚到足够的银子后，终于要启程逃往国外了。很不幸，不知怎么回事，HYC准备流亡海外的计划被天朝的官员得知了。由于早先HYC无情地鄙视了他们的智商，他们决定打击报复，在将HYC抓捕归案的同时贬低一下TA的智商，以一雪前耻并为平淡无奇的日常生活寻找一些乐趣。于是他们下令封锁所有出境的港口和机场，对于所有要出国的乘客，进行智商测试，通过者才能放行。很明显，天朝官员认为HYC是不可能通过智商测试的，而这也正是他们喜闻乐见的结果。

智商测试是这样的：对于每个想要出国的乘客，官方会随机给出一个长度为n的字符串，全部由小写字母组成。定义一个字符串的价值为其中出现次数最多的字母出现的次数减去其中出现次数最少的字母出现的次数，比如aaaba的价值为4-1=3。当然，天朝官员毕竟是通过公务员考试才进入政府机构的，他们不可能出询问一个字符串的价值如此白痴的问题。为了加大测试的难度以真正达到测试智商的目的，他们要求每个乘客在1s的时间内回答出一个字符串所有子串中价值最大的子串的价值。

很不幸，轮到HYC回答问题的时候，HYC正好处在TA每个月的那几天低谷期，所以智商瞬间跌到和天朝官员一个水平了。无奈之下，LZT只好再度出手，暗中将问题的答案告知HYC。


输入
输入文件第一行包含一个整数n，表示字符串的长度。第二行包含n个小写字母，描述了该字符串。
输出
输出文件包含一行仅一个整数，表示输入中字符串的所有子串中价值最大的子串的价值
样例输入
10
aabbaaabab
样例输出
3
提示
价值最大的子串为aaaba，价值为4-1=3。
对于10%的数据，1≤n≤100。
对于30%的数据，1≤n≤2500。
对于100%的数据，1≤n≤1000000。
