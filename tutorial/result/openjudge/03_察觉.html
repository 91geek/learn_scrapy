描述

萝卜的奇袭不幸被lj发现了，于是萝卜引大军当主lj主力，只让关羽奇袭基地。已知关羽的为17m/s，以这样的速度是无法奇袭成功的。庆幸的是关羽拥有技能——千里行，可在1s内移动 60m，不过每次使用千里行都会消耗采配点10点。关羽采配点的回复速度为4点/s，只有处在原地休息状态时才能恢复。
　　现在已知关羽的采配点初值M，他所在的初始位置与地方基地之间的距离S，敌人撤回基地时间T。你的任务是写一个程序帮助守望者计算如何在最短的时间内发动奇袭，若不能奇袭，则输出关羽在剩下的时间内能走的最远距离。注意：所有运动均以秒(s)为单位，且每次活动的持续时间为整数秒。距离的单位为米(m)。


输入
仅一行，包括空格隔开的三个非负整数M, S, T。
输出
包含两行：　　第1行为字符串“Yes”或“No”（区分大小写），即关羽是否能成功奇袭。　　第2行包含一个整数。第一行为“Yes”（区分大小写）时表示关羽奇袭成功的最短时间；第一行为“No”（区分大小写）时表示关羽能走的最远距离。
样例输入
样例1：
39 200 4
样例2：
36 255 10
样例输出
样例1：
No
197
样例2：
Yes
6
提示
数据规模和约定
　　30%的数据满足：1 <= T <= 10, 1 <= S <= 100　　50%的数据满足：1 <= T <= 1000, 1 <= S <= 10000　　100%的数据满足：1 <= T <= 300000, 0 <= M <= 1000, 1 <= S <= 100000000.
