描述

别以为题目说Fruit Ninja就是切水果，我们玩的是砸水果，不是吃水果，也不是卖水果，是砸水果哟~用板砖砸哟~

屏幕上有N个物品（点），可能是水果，可能是炸弹，现在我们用一个高为H，宽为W的板砖（矩形）去框水果，不过可不能框到炸弹哟~如果恰好某个物品落在矩形边界，也算框住的说。

途中必须保证长为W的边始终平行x轴，长为H的边始终平行y轴。

我们的目标是框住最多的水果，不过你需要回答有多少种方法达到这一目标，只要板砖（矩形）的位置不同就算不同的方法。


输入
每组数据第一行将是三个整数N H W，分别代表物品总数，板砖的高和宽。
接下来N行每行的格式是 L X Y，L是一个字符，如果是'G'，代表水果,如果是'B'，代表炸弹。X，Y是这个物品的坐标，没有两个物品坐标相同。
1<=N<=50000, 1<=X,Y,H,W<=25000.
输出
第一行，如果方法数有限，输出这个值，否则输出"so many!".
第二行，输出最多能够框住的水果数目
样例输入
4 2 3
G 0 1
G 0 2
G 1 2
G 3 0

4 2 3
B 0 1
B 0 2
G 1 2
G 3 0

4 2 3
B 0 1
B 1 2
G 1 1
B 3 0
样例输出
1
4

so many!
2


so many!
