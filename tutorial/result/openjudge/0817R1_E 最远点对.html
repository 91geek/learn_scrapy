0817R1:E 最远点对

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
单个测试点时间限制:
100ms
内存限制:
65536kB
描述
对于给定的n个点，请你求出相距最远的两个点。
输入
多组测试数据，每组测试数据中，
第一行：n,代表点数(2<=n<=50000)
以下n行：每行两个数，xi,yi，表示第i个点的坐标(-10000 <=xi,yi<= 10000)
以文件结束符结束。
输出
最远点对的距离的平方
样例输入
4
0 0
0 1
1 1
1 0
样例输出
2
