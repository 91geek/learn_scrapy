0009:2014-10-04 咏絮才

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
262144kB
描述

hj和jh在愉快地玩耍。他们在一张n个点m条边的无向图上观赏风景。
jh想知道，存不存在一种游览方案，从一个点出发，刚好经过每条边一次，最后回到起
点。（连接相同两个顶点的可能有多条边，这时每条边都必需被走一次）

输入
每个测试点包含多组数据。
第一行为一个正整数t，代表数据组数。
对于每一组数据，第一行是两个正整数n和m，表示点数和边数。接下来m行每行有两
个正整数u, v，表示从u到v有一条无向边。
输出
输出共t行。如果第i个测试点的图存在如题的方案，这一行输出"TAK"，否则输
出"NIE"。（不含引号）
样例输入
2
1 0
3 2
1 2
2 3
样例输出
TAK
NIE
提示
对于50%的数据，1 ≤ t ≤ 5, 1 ≤ n ≤ 10, 0 ≤ m ≤ 20。
对于100%的数据，1 ≤ t ≤ 100, 1 ≤ n ≤ 1000, 0 ≤ m ≤ 3000。
对于100%的数据，1 ≤ u, v ≤ n，不保证没有重边和自环。
