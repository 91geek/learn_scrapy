04:Eating Cheese

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

Jerry is going to steal Tom's cheese. All the cheeses lined up in a straight line, each with a different deliciousness. However, if Jerry steals two adjacent pieces of cheese, Tom will find this and quickly grab Jerry. Jerry hopes to enjoy the maximum sum of deliciousness without being discovered by Tom. Of course, he can also choose not to eat any piece of cheese. Please help him make a plan for stealing food and tell him how much deliciousness he can steal at most.


输入
The first line has an integer T (T<=100), indicating the number of test cases.

Next, each test case contains two lines. The first line contains an integer n (1 <= n <= 100,000), which represents the number of cheese. the second line contains n integers, represents the deliciousness of each cheese on this row of lines. Please note that the deliciousness can be stored using int data type and it can be negative.
输出
For each test case, the output is an integer, indicating the maximal sum of deliciousness that Jerry can obtain.
Please note that the sum of deliciousness may exceed the range of integer.
样例输入
2
4
1 2 3 1
5
2 7 9 3 1
样例输出
4
12
