描述

打完比赛，天色渐晚，小 Y 和小 Z 决定结伴回家，以防不测。不料，一路上的建筑物模样愈来愈奇怪， 最终竟变成了一个又一个的数字。只见那些数字一开始还是歪歪斜斜，七零八落地散在角落里，不一会就仿佛被一股神奇的力量操纵着般整整齐齐地聚集在了一起，形成了像平顶山一样硕大的矩形，好不壮观！小 Y 和小 Z 吓得赶紧抱在了一起，紧张地抽泣着。就在这时，一个低沉而又富有磁性的声音出现了。「欢迎来到矩阵的王国！」，那个声音说道。「我们的王国最近遇上了一件烦心事，如果你们帮我解决这个问题，我就立即送你们回家。要是解不出来，吼吼，我就让你们线代老师挂了你们。」听到事情的原委，小 Y 和小 Z 顿时舒了一口气，可是听到后面自己的线代可能要挂，小 Y 和 小 Z 的心一下子又提到了嗓子眼。聪明的你快来帮帮小 Y 和小 Z，解决矩阵王国的难题，帮他们保住线代！

输入
第一行为两个整数 N, P。1 ≤ N ≤ 20; 0 ≤ P ≤ 100000
之后的 N 行表示一个 N 阶方阵 A，其中第 i (1 ≤ i ≤ N) 行包含 N 个整数，分别表示 a_ij（1 ≤ j ≤ N; 0 ≤ a_ij ≤ 2000000000）。
输出
输出将矩阵 A^P（A 的 P 次幂，即 P 个 A 乘在一起）中每一元素对 1000000007 取模后的矩阵。
样例输入
2 4
1 2
3 4
样例输出
199 290 
435 634
提示
对于 n × m 矩阵 A 与 m × p 矩阵 B 的乘积 C = AB，定义 c_ij = Σ a_ik⋅b_kj (1 ≤ k ≤ m)
