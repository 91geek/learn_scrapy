2-1-5分数等级查询小系统

    我们知道老师一般会把分数进行分级，学生可以根据自己的考试分数来查询自己的成绩是哪个等级：    

80 < score <= 100 为优秀60 <= score <= 80 为良好score < 60 为不及格

既然我们知道了分等级的规则就帮着老师来做一个分数等级查询的小系统吧。

例如：优秀等级对应的C语言的语句是：

    if (score <=100 && score >80)

    {

    printf("good\n");

    }

课堂作业：输入一个0-100的整数，判断它的分数等级。

如果80< score <= 100 要打印出："good"

如果60<= score <= 80 要打印出："pass"

如果score< 60 要打印出："fail"

如果超出0-100的范围，无输出

参考2-1-5视频教程最后的代码进行修改，提交到openjudge11中题库。此代码可使用条件分支if和else语句的嵌套。


输入
输入一个整数
输出
80 < score <= 100 要打印出："good"
60 <= score <= 80 要打印出："pass"
score < 60 要打印出："fail"
如果超出0-100的范围，无打印输出
样例输入
80
样例输出
pass
提示
提示：题目要求输出的英文单词可以复制哦，免得自己输错
