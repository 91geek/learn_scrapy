t4:【T4】陨石坠落

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
131072kB
描述

小可可听说了一个骇人听闻的消息：陨石风暴即将袭击她所在的城市，由于陨石体积过大，它们无法在撞击到地面前燃烧殆尽，届时将会对它撞到的一切东西造成毁灭性的打击。小可可开始担心自己的安全问题。她一定要在被陨石砸到前，到达一个安全的地方（也就是说，一块不会被任何陨石砸到的土地）。如果将她所在的城市放入一个直角坐标系中，小可可现在的位置是原点，并且，小可可不能踏上一块被陨石砸过的土地。

根据预报，一共有M颗陨石(1 <= M<= 50,000)会坠落在城市上，其中第i颗陨石会在时刻T_i (0<= T_i <= 1,000)砸在坐标为(X_i, Y_i) (0 <= X_i <= 300；0<= Y_i <= 300)的格子里。陨石的力量会将它所在的格子，以及周围4个相邻的格子都化为焦土，当然小可可也无法再在这些格子上行走。

小可可在时刻0开始行动，它只能在第一象限中，平行于坐标轴行动，每1个时刻中，她能移动到相邻的（一般是4个）格子中的任意一个，当然目标格子要没有被烧焦才行。如果一个格子在时刻t被陨石撞击或烧焦，那么小可可只能在t之前的时刻在这个格子里出现。

请你计算一下，小可可最少需要多少时间才能到达一个安全的格子。

输入
第1行: 1个正整数：M
第2..M+1行: 第i+1行为3个用空格隔开的整数：X_i，Y_i，以及T_i
输出
输出1个整数，即小可可逃生所花的最少时间。如果小可可无论如何都无法在陨石风暴中存活下来，输出-1
样例输入
4
0 0 2
2 1 2
1 1 2
0 3 5
样例输出
5
提示
样例说明请参考pdf题面第6页。
