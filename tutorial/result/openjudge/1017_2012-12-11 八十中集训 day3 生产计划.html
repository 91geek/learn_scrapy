1017:2012-12-11 八十中集训 day3 生产计划

  • 查看
  • 提交

总时间限制:
20000ms
单个测试点时间限制:
1000ms
内存限制:
1280000kB
描述

小Y很喜欢汽车，经过奋斗，开了一家汽车厂。厂中共有N个工人。他们在
流水线上批量生产汽车。工人们被编号为1到N。每个工人都有专门负责的工作，
并且需要特定的时间完成。

一辆车的生产过程是这样的。首先从工人1开始，完成他负责的部分，然后
由工人2接手，继续完成工人2负责的部分，然后交给工人3...以此类推，直
到工人N完成了他负责的部分以后，这辆车的生产过程就结束了。

现在，小Y的工厂需要按顺序完成M辆汽车的制造。对于每个工人i，我们
知道他完成他负责的部分所需要的基本时间，记为Ti。这M辆汽车中的每辆汽
车j都有一个参量Fj，表示这辆车的技术复杂程度。对于第j辆汽车，我们知
道工人i在汽车j上完成他负责部分的时间应该等于Ti*Fj。

在流水线上，任何一个工人在完成他负责的部分以后，都会立即（毫不延迟
地）将汽车移送给下一个工人。因此，每个工人在接受一辆汽车时必须是空闲的，
否则会发生事故。于是，小Y必须选择合适的时间开始一辆新汽车的生产。为了
追求高效率，小Y会等待尽量少的时间，直到她认为下一辆车的生产条件已经得
到满足，就立刻开始生产。

现在，告诉你相关的信息，请你计算出这M辆车的生产所需要的时间。

输入
输入文件的第一行包括两个正整数N,M，分别表示工人的数目，和汽车的数
目。
接下来的N行，每行1个正整数。依次表示T1,T2,.., TN。
接下来的M行，每行1个正整数。依次表示F1, F2, .., FM。
输出
输出文件仅包括1行，为1个整数，表示M辆车的生产过程需要的时间。
样例输入
3 3 
  2 
  1 
  1 
  2 
  1 
  1
样例输出
11
提示
100%的数据满足，1 <= N <= 100,000, 1 <= M <= 100,000, 1 <= Ti, Fi <= 10000.
