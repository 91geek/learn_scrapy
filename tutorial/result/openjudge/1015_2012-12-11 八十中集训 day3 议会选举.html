1015:2012-12-11 八十中集训 day3 议会选举

描述

又到了议会选举的时间。这次有共计V个选民参与投票，每一个选民将会把
选票投给N个政党之一。议会中有M个席位即将选民们被选出。

不妨将这N个政党从1到N编号，假设它们依次获得了V1, V2, ..., VN张选
票。议会的席位将通过以下的过程分配：

1. 得票数严格小于V的5%的政党将不具有候选资格，即不会获得议会的任
何席位。（以下“政党”均指具有候选资格的政党）

2. 最初每个政党的席位数都等于0。

3. 对于每个政党p，设Vp表示政党p获得的选票数，Sp表示政党p已经获
得的席位数。我们计算参量Qp=Vp/(Sp+1)。

4. 当前参量Qp 最大的政党将会获得一个新的席位。若有多个政党参量相同，
那么选择编号尽量小的那个政党。

5. 重复步骤3和步骤4直到议会中所有席位均被分配完毕。

现在，选举仍然在进行中，仅有部分选票得到统计。我们已经知道这N个政
党当前获得的选票数。

小Y想请你帮助她，计算对于每个政党，在所有可能的选举结果中，它可能
获得的最多席位数和最少席位数。

输入
输入文件的第一行包括三个正整数V, N, M，分别表示选民数（即总选票数），
政党数，议会的席位数。
第二行包括N个非负整数，依次表示当前N个政党分别获得的选票数。保证
这些数之和不大于V。
输出
输出文件的第一行包括N个整数，用空格隔开，第i个整数表示第i个政党
最终可能获得的最多席位数。
第二行包括N个整数，同样用空格隔开，第i个整数表示第i个政党最终可
能获得的最少席位数。
样例输入
20 4 5 
  4 3 6 1
样例输出
3 3 3 2 
1 0 1 0
提示
100%的数据满足，1 <= V <= 10,000,000, 1 <= N <= 100, 1 <= M <= 200.
