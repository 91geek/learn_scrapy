1033:2014-11-21 军队调遣

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
262144kB
描述

OI 国有N 座城市，每座城市有唯一的1 到N 的标号。其中1 号城市为首都。OI 国的交通
十分发达，共有M 条有向道路，已知通过第i 条道路所需时间为ti。
林大帝的统治使OI 国走向了繁荣。
不幸的是，各地都仍旧有一些不老实的人在捣蛋。林大帝实在是不能忍了。为了加强治安，
保障国土的完整，林大帝派人在每座城市（除了首都本身）招募军队去剿匪。匪徒都是些乌合
之众，没过一会儿就被林大帝的精锐之师打得落花流水、心服口服。
为了庆祝剿匪胜利，各地的军队都必须尽快到首都集合。然而，行军是会产生劳累值的，且
军队在剿匪之后十分疲惫，所以他们会优先选择劳累值最小的路径。如果有多个劳累值最小的
路径，他们会在其中选择用时最少的路径。
当军队经过一条u 到v 的道路时，如果v 能够到达u，士兵们就会有绕圈的感觉，故产生1
单位的劳累值；否则，士兵们会有说有笑地前进，不会产生劳累值。一条路径的劳累值为路径
上所有边的劳累值之和，一条路径的用时为路径上所有边的用时之和，如有边重复经过需累计。
你需要对每座城市计算其军队将选择的路径的劳累值及用时，或指出无法到达首都。

输入
第一行两个正整数N;M，分别为城市数量和道路数量。
接下来M 行每行描述一条道路：ai; bi; ti 表示一条从ai 连向bi 且用时为ti 的道路。
输出
输出共N - 1 行，第i 行描述i + 1 号城市的军队选择的路径情况。如果该城市的军队不能
到达首都，输出-1；否则输出两个整数依次表示劳累值和用时。
样例输入
3 3
1 2 1
2 3 1
3 1 1
样例输出
2 2
1 1
提示
对于20% 的数据，N;M <= 20。
对于50% 的数据，N;M <= 2000。
对于100% 的数据，1 <= N;M <= 10^5; 0 <= ti <= 10^9。
