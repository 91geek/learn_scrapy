02:交换

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
1000kB
描述

亚里士多德说过这样一句话：

如果我有一个苹果，你有一个苹果，交换一下，还是你有一个苹果，我有一个苹果。

假如我有一种想法，你有一种想法，交换一下，每个人都有了两种想法。

将下面的代码补充完整，实现两个数交换。

		
		
// 在此处补充你的代码
		
				
int main()
{
	int a,b;
	scanf("%d",&a);
	scanf("%d",&b);
	printf("%d,%d\n",a,b);
	swap(&a,&b);
	printf("%d,%d\n",a,b);
	return 0;
}
		
输入

		
两个整数，比如：
2
3

		
输出

		
交换前及交换后的两个整数，中间用逗号隔开，比如：
2，3
3，2

		
样例输入

		
123
456

		
样例输出

		
123，456
456，123
