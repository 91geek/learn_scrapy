描述

小 P 在玩一个游戏,首先他有一个序列 a

每次他可以从 a 的左边拿一个数,或者 a 的右边拿一个数,那么可以得到序列 b

如果存在一个序列 b 使得 b0*50+b1*51+....+bn-1*5n-1=3(mod 8)

那么称 a 序列为好序列

玩了一会儿发现太简单了,于是小 P 想在一颗树上玩

一颗 n 个点的数,每个点有一个权值

F(A,B)表示有 A 到 B 的路径上的点权组成的序列

小 P 认为若 F(A,B)是好序列且 F(B,C)是好序列,那么 F(A,C)也是好序列

或者若 F(A,B)不是好序列且 F(B,C)不是好序列,那么 F(A,C)也不是好序列

想在想知道小 P 正确的情况有多少种?即有多少对(A,B,C)小 P 的结论是正确的?

注意(A,B,C)与(B,C,A)只算一次
输入
第一行一个数 n
接下来 n 行每行两个数 Fi,Vi 表示 i 的父亲是 Fi,i 节点的值是 Vi (1<=i<=n)
若 Fi=0 表示 i 节点是根节点
输出
输出一行表示答案
样例输入
3
0 3
1 5
1 7
样例输出
0
提示
数据范围
30%
n<=1000
100%
n<=100000 0<=Vi<8
