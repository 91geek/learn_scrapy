描述

“0/1字符串”指的是所含字符要么是“1”，要么是“0”的字符串。0/1字符串有两个操作:

     * switch :1变成0,0变成1 。比如“100”，变成“011”

     * reverse:字符串反过来，比如"100"，变成“001”

考虑如下无限0/1字符串：

S0=""

S1="0"

S2="001"

S3="0010011"

s4="001001100011011"

...

S(N)=S(N-1)+"0"+switch(reverse(S(N-1))

请输出S(googol)的第K个字符，其中googol=10^100

输入
第一行表示测试数据数量T
其后的每一行包含一个整数K
输出
对于每个测试案例，输出"Case #x:y",其中x指测试案例编号（从1开始），y指S(googol)的第K个字符
样例输入
4
1
2
3
10
样例输出
Case #1: 0
Case #2: 0
Case #3: 1
Case #4: 0
提示
数据限制：
1 ≤ T ≤ 100.
1 ≤ K ≤ 10^18
