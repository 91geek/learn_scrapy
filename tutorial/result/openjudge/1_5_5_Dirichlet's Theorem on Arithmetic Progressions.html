1_5_5:Dirichlet's Theorem on Arithmetic Progressions

  • 查看

总时间限制:
1000ms
内存限制:
102400kB
描述

如果a和d是互素的正整数，从a开始增加d的算术序列，即a,a+d,a+2d,a+3d,...,包含无穷多的素数。这被称为Dirichlet算术级数定理。（具体可以百度一下）

给出正整数a、d和n，写一个程序给出算术序列中第n个素数。

输入
输入是一个数据集合的序列，一个数据集合一行，包含了3个用空格分开的正整数a，d和n。a和d互素。设定a < = 9307 ,d < = 346, n < = 210
输入用3个空格分开的0结束。
输出
输出行于输入的数据集合的行数相同。每行输出一个整数，对应的是从a开始每次增加d的算术序列中的第n个素数。
结果肯定在整数范围内。
样例输入
367 186 151
179 10 203
0 0 0
样例输出
92809
6709
提示
互素，又称互质。
若n个整数a1,a2,…,an的最大公因数为1，就称这n个整数互素．
