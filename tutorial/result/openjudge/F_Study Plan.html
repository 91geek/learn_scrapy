F:Study Plan

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

Definiter is preparing for the TOEFL. He has to memorize oceans of words. As we all know, it is not enough to learn a word only once without reviewing. However, due to the limited time, Definiter decides to make a study plan so that each word can be learned once and reviewed once.

More scientifically, for some words, it is better to review them in two days after the first learning, while there are also some other words that being reviewed after a month would enhance the memory. In other words, each word has different Best Memory Interval (BMI), which describes the best memory retention of a word -- how many days should exactly be between the first learning and its next reviewing (not including these two days). For example, if Definiter learns a word with BMI-1 today, then he had better review it the day after tomorrow, so that the interval is exactly one day.

Definiter divides all the words into N parts according to their BMI from 1 to N. Every word in part i has BMI-i. He wants to make a perfect study plan, which satisfies:

  1. Definiter does not want to waste time. He has to either learn or review one part every single day. No rest day is allowed. And every part should be exactly learned once and reviewed once, which implies that the plan contains exactly 2N days.

  2. Definiter wants to follow every word's BMI. For part i, the interval between learning and reviewing should be exactly i days.

But Definiter finds it hard to make a perfect study plan. He wonders if there is no such plan. He needs your help :).

输入
The first line contains an integer T (1 ≤ T ≤ 10) -- the number of test cases.

For each test case, there is only one line containing an integer N. 2 ≤ N ≤ 10000.
输出
For each test case, output 2N numbers in a single line -- a permutation of 1, 1, 2, 2, ..., N, N, represents a perfect study plan (which part to be learned/reviewed on each day), separated by a single space. If there are several solutions, output any one of them. If there is no such plan, only output a “0” in a single line.
样例输入
3
1
2
3
样例输出
0
0
2 3 1 2 1 3
