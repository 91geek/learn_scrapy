F:辅导员再点名

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
256000kB
描述

最近，佛大开展校风、学风建设活动。各学院辅导员收到上级通知，各自采取不同的手段对找出迟到和旷课的同学并进行批评。于是电XX信XX学院的辅导员杨胖子想到了一个妙法——亲自出马点名。他的点名方法和其它的老师不同，他是按学号和名字交替的顺序进行抽样检查T个学生。这让我等学渣情何以堪。

例如：T = 5时

5

abcd

4

bac

3

第1、3、5行是学生的学号，2、4行是学生的名字。

另外，由于杨胖子记性不好，他有时会忘记哪些学生已经被点过名了，所以会出现重复点名的情况。

5

abcd

5

bac

abcd

5号同学被点了两次，abcd也被点了两次。

在计算机某班，假定该班有N个学生，每个学生都有一个唯一的学号和姓名。很不幸运，杨胖子今天按照他的聪(yin)明(xian)方法抽点了该班T名学生，这让没来上课的学渣们痛苦不已。(*_* ) (*_* ) (*_* ) 。作为一名老学渣总是怀有侥幸心理，不关心谁来没来上课，只关心自己有没有被点名。(^o^)哇~~(^0^)哈~~(^○^)哈~~~

现在他们要作为副班长的你，提供未被点名的学生的名单。

输入
第一行输入一个0 < N <= 400000，代表有N个学生。

接下的N行是学生的名字，每个学生的名字都是由小写字母组成，并长度不超过4。这些输入的名字按学号从1到N的顺序自动编号。

接着是输入0 < T <= 400000,代表被点名的学生的个数。

然后按照学号和名字交替输入T条被点名的记录。（先输入学号）
输出
在未被点名的学生名字中，每隔100个名字输出一次，每个名字占一行。如果没有人未被点名，则输出”none”
注意：是每100个名字输出一个名字，不是输出第一百个名字。例如：如果未被点名的学生共110个，则输出地1个和第101个；如果有1010个学生，则输出第1、101、201、301、401、501、601、701、801、901、1001个未被点名学生的名字。
样例输入
15
a
b
c
d
e
f
g
h
i
j
k
l
m
n
o
3
1
b
3
样例输出
d
