1001:2014-09-23 合成

  • 查看
  • 提交

  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
262144kB
描述

Mr.Lin 有N 种材料，第i 种材料的硬度为Hi。现在，他把它排成一行，第i 种放在第i 位。
现在，他要选择一段连续的区间（共有N(N+1)/2 个不同的区间），将区间中的所有材料合成
为新的材料。硬度在合成时显然是会发生变化的。具体来说，将硬度为a 和b 的材料合成后
新材料的硬度为a or b。求对所有区间进行合成得到的硬度中第K 大的硬度。

输入
第一行一个正整数N 表示材料数目。
第二行一个正整数K。
第三行N 个正整数Hi。
输出
输出一行一个整数表示答案。
样例输入
2
2
1 2
样例输出
2
提示
数据约束
对于20%的数据，N <= 100
对于40%的数据，N <= 1000
对于70%的数据，N <= 100000
对于100%的数据，1 <= N <= 500000，0 <= Hi <= 2^31-1, 1 <= K <= N(N + 1) / 2
