C:Binary Search

总时间限制:
1000ms
内存限制:
65536kB
描述

We all know binary searching, it is simple and powerful.
But do you really understand it?
Here I am going to challenge you with a binary searching problem.

Given a array of N integers, sorted into non-decreasing order,
and M queries, each query has following format:

    T X

where both T and X are integers, T represents the type of operation:

    0: print the index of the first element which is not less than X.
    1: print the index of the first element which is greater than X.
    2: print the index of the last element which is less than X.
    3: print the index of the last element which is not greater than X.

输入
Input starts with N (1 <= N <= 100000) and M (1 <= M <= 1000).
The next line contains the array of N integers, where the
absolute value of those integers is not greater than 10000.
The following are M query lines as described above.
输出
To each query, print the index it asks.
样例输入
7 8
2 3 3 3 5 5 7 
0 8
1 8
2 1
3 1
0 3
1 3
2 3
3 3
样例输出
7
7
-1
-1
1
4
0
3
提示
Operation "0 3" and "1 3" can form a range [L, R) in order to
iterate all the 5's sequentially,
in this case: L = 1, R = 4

Operation "2 3" and "3 3" can form a range (L, R] in order to
iterate all the 5's sequentially in reversed order,
in this case: L = 0, R = 3
