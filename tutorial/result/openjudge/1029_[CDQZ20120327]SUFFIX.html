描述

本题中的字符串只包含'a'到'z'的字符，例如“abcdefg”, “isun”是合法的，而“rocket323”是不合法的。


某字符串的后缀suf[i]是指从i到n-1的所有字符(字符串的下标从0到n-1，n为字符串长度）。


某个字符串的后缀表是指0~n-1 这n个数的一个排列L（n为字符串长度），并且满足


suf[L[0]] < suf[L[1]] < suf[L[2]] < …< suf[L[n-1]].


例如对于字符串"aabac", 它的后缀表是[0,1,3,2,4]


给定一个字符串S和一个后缀表L，求最少需要改变多少个字母，才能使得新的字符串的后缀表恰好为L。


输入
对每组数据，第一行是一个n，代表字符串长度 (1 <= n <= 500)
第二行是字符串
第三行是一个 0~n-1的排列
输出
输出最小的改变数，如果无解输出-1
样例输入
3
abc
0 1 2

3
aab
2 0 1
样例输出
0

2
