1082:Cvvvvv和ACM

描述

据说当年Cvvvvv进入我校ACM队的过程可谓一路坎坷。当他去联系已是ACM队员的ACSync表达自己的意愿时，ACSync什么都没有说，只是给了他几道题让他回去做，做出来就可以进入校ACM队。但是当时的Cvvvvv并没有任何做算法题目的经验，那些题目对于当时的他来说非常困难，所以他只好找他的基友czvino帮他完成这些题目。czvino很快地帮助他完成了大部分题目，但有一道题让czvino也犯难了起来，请问你能帮助他们完成这道题目吗？题目描述如下：

       一个包含 l 个数字的数组b1, b2, ..., bl (1 ≤ b1 ≤ b2 ≤ ... ≤ bl ≤ n) 被称作是好数组当且仅当数组中的每一个数都能整除它后一个数字，即bi+1 % bi= 0 (1 ≤ i ≤ l - 1)。

       给定n和k，请给出长度为k的好数组的个数。由于答案可能过大，请给出其对1000000007 (109 + 7)取模的结果。


输入
第一行包含两个整数n和k (1 ≤ n, k ≤ 2000)，两数之间以空格隔开
输出
输出一个整数，表示长度为k的好数组的个数对1000000007取模的结果。
样例输入
3 2
样例输出
5
提示
对于第一组示例，好数组为[1，1]，[2，2]，[3，3]，[1，2]，[1，3]。
