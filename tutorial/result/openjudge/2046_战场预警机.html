战场预警机

　　所谓预警机，就是一种装有雷达的，可以在空中为我方战斗机提供战场信息的军用飞机。
　　现在，C国要开发一套系统，使得预警机上的敌我目标信息以图形化的方式显示出来，并且能够接受指令来添加、删除新目标。

　　图形显示方式为 60列*20行 字符阵列，敌我目标的坐标将被映射到[0..19][0..59]的范围内，最新添加的目标显示在最下面，会被旧目标覆盖。
　　空地处用字符' '填充。
　　我方目标：

_
<_>

　　敌方目标：

\ /
X
/ \


所有的坐标信息保存在一个链表里面，你可以使用如下结构体来保存坐标信息：

typedef struct TypePosition {
int x, y, t;
struct TypePosition *next;
} Node, *PNode;


有以下5种命令：
insert x y t 表示向链表最后插入标记(x,y)，t=0表示我方，t=1表示敌方。
delete x y 删除链表中的标记(x,y)，如果有的话。
moveto ax ay bx by 位于(ax,bx)的标记移动到(bx,by)
display 更新图形显示
exit 退出程序


　　请编写delete命令和moveto命令的代码，其他部分的代码（例如display部分）可以直接复制现有的框架，无需自己编写。输入用例将保证同一时刻不会有2个目标在一个坐标上。


=====================================

typedef struct TypePosition {

return x >= 0 && x < 20 && y >= 0 && y < 60;
}

p->x = x;
p->y = y;
p->t = t;
p->next = pt->next;
pt->next = p;
}

int main() {
int x, y, xt, yt, t;
int i, j;
char display[20][60];
char buf[0x10];

// 创建链表
PNode list = (PNode)malloc(sizeof(Node)), p;
list->next = NULL;

// 主循环
while(scanf("%s", buf), strcmp(buf, "exit")) {
if(strcmp(buf, "insert") == 0) {
scanf("%d%d%d", &x, &y, &t);
insert(list, x, y, t);
}
else if(strcmp(buf, "delete") == 0) {
scanf("%d%d", &x, &y);
deletes(list, x, y);

for(p = list; p->next; p = p->next)
if(p->next->t) {

if(inarea(p->next->x - 1 + i, p->next->y - 1 + j) && sg1[i][j] != ' ')
display[p->next->x - 1 + i][p->next->y - 1 + j] = sg1[i][j];
}

}
