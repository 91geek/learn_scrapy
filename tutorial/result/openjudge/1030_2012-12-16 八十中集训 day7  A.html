总时间限制:
200000ms
单个测试点时间限制:
1000ms
内存限制:
2560000kB
描述

给一个n个点m条边的无向图，每个点要么是黑色，要么是白色，需要完成q个操作，操作有
两种：
• Change x：改变x号点的颜色，黑色会变成白色而白色会变成黑色。
• Asksum a b：求所有满足一端点颜色为a而另一端点颜色为b的边的权值和。
写个程序来完成这些操作。

输入
第一行两个正整数n和m。
第二行是n个用空格隔开的整数，第i个数表示第i个点的颜色，0表示黑色，1表示白色。
接下来有m行，每行三个正整数u，v，w，表示u和v之间有一条权值为w的边。
接下来一行一个正整数q。
最后有q行，每行一个命令，格式如题目所述
输出
对每个“Asksum”操作，输出一行一个整数，表示答案。
样例输入
4 3
0 0 0 0
1 2 1
2 3 2
3 4 3
4
Asksum 0 0
Change 2
Asksum 0 0
Asksum 0 1
样例输出
6
3
3
提示
对于100%的数据，满足1<=n,m,q<=10^5 1<=w<=2^31-1
