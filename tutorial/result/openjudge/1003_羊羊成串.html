1003:羊羊成串

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

现有总数为N的一群羊，每只羊都有一个数字Ai（1≤i≤N），现在你需要把这N只羊排成一串（也即一行），使得由它们的数字串接起来的数字最大。

比如，现在有3只羊，它们的数字分别是1，2和3，则它们串接起来的数字可以是：123、132、213、231、312和321共6种情况，其中最大的是321。

现在，你的任务就是输出最大的那个数字。

输入
第一行包含一个整数T，表示测试数据的组数。
每一组测试数据包含两行，第一行为整数N，表示羊的数量；接下来的一行依次包含N个整数，分别表示每个羊的数字（整数）。
数据范围：
50%：1 <= T <= 20，1 <= N <= 20，0 <= Ai <= 100 (1 <= i <= N)；
100%：1 <= T <= 20，1 <= N <= 5*10^4，0 <= Ai <= 10^9 (1 <= i <= N)。
输出
对于每一组测试数据，输出测试序号（从1开始），以及对应的最大的整数。
样例输入
3
3
1 2 3
3
1 1 9
3
92 91 99
样例输出
Case #1: 321
Case #2: 911
Case #3: 999291
提示
测试点2个，各15分，共30分。
