1037:[CDQZ20120411]First

总时间限制:
25000ms
单个测试点时间限制:
5000ms
内存限制:
262144kB
描述

        已知一些点，每个点由横坐标和纵坐标描述。

        现有如下各种操作，具体如下：

        0.将所有点平移

        1.将所有点放大或缩小

        2.将所有点以X轴翻转

        3.将所有点以Y轴翻转

        4.将所有点以原点旋转

        5.加入一个新的点

        6.查询某个点当前的坐标

        #点的编号由1开始#

输入
第1行2个整数,n,m，分别表示起始时点的个数和操作次数
第2行 至第n+1行，每行2个实数，表示点的坐标
第3行至第m+2行，每行一个整数k，表示操作类型，如题目描述所示。对于0，1，5操作，将再读入2个实数（平移的向量，X坐标和Y坐标需要扩大的倍数，加入点的坐标）；对于4操作，将再度如1个实数（旋转的弧度）；对于6操作，将再读如1个整数（点的编号）
输出
输出询问操作所对应的点的坐标（保留0位小数）。
样例输入
2 3
0 0
1 1
0 2 3
6 1
6 2
样例输出
2 3
3 4
提示
1<=n<=1000000，1<=m<=3000000
