1018:n位数逆序

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
32000kB
描述

输入一个 n (1 <= n < 10) 位数，输出它的逆序数码。

倘若输入的数字是 100000，则应该输出 000001.

输入
第一行一个整数 T，代表数据组数。
对于每组数据只有一个整数 a (1 <= a < 1 000 000 000)，且不含前导 0
输出
对于每组数据输出一行，为整数 a 逆序后的数码
样例输入
5
314
327
52000
100000
23333333
样例输出
413
723
00025
000001
33333332
