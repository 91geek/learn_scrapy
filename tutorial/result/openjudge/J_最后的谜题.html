描述

定义f(a,n)={a^k mod n | k是正整数}，表示a在模n下生成的集合。

|f(a,n)|表示这个集合的元素个数。

对于给定的a和n，试解决以下两个问题：

1.计算|f(a,n)|

2.计算方程f(x,n) = f(a,n)在[1,n-1]的范围内有多少个整数解

输入
第一行一个整数a，第二行一个整数n。
保证1 <= a < n <= 5*10^5
输出
第一行一个整数表示问题一的答案。
第二行一个整数表示问题二的答案。
样例输入
7
15
样例输出
4
2
提示
f(7,15) = {7,4,13,1}
f(4,15) = {4,1}
f(13,15) = {13,4,7,1}
f(1,15) = {1}
所以方程f(x,15) = f(7,15)有两个解：7、13。
