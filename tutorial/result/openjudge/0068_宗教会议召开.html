0068:宗教会议召开

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
256000kB
描述

解放区以往的宗教会议因福音的传播而其乐融融，但自从全能神和观音法门两个极具蛊惑性的邪教出现后，宗教会议变味了。一些村民误信邪教，导致村里乌烟瘴气。中央经调查发现一个含n个村庄的地区竟然全被邪教荼毒。为了打击邪教，中央准备从邪教召开的会议入手进行治理。

这个地区的村庄间通过n-1条道路相连接，同一个村的村民都信同一个邪教，每当一个邪教要召开会议时，信该教的教徒都会去参加。但因为两个邪教之间也存在竞争，一个邪教领导的村庄是不会允许异教徒经过这里去参加会议的。现在中央想知道当某个邪教在一个村庄召开宗教会议时，有多少个村庄的村民会来参加。

有时一个村庄的村民会被群体蛊惑，于是全部改信另一邪教。

一开始这片地区的所有村民均误信邪教全能神。


输入
第一行两个整数n，m，表示村庄数和操作数。
接下来n-1行，每行两个整数x，y，表示x和y之间有一条道路。
接下来m行，每行两个整数k，x：
如果k=0，表示y村庄召开宗教会议，问能召集多少村庄的村民。
如果k=1，表示y村庄的村民均误信了另一邪教。
输出
对于每个询问，输出一行一个整数，即有多少村庄的村民参会。
样例输入
3 5
1 2
2 3
0 1
1 2
0 1
1 3
0 3
样例输出
3
1
2
提示
【数据规模】
对于30%的数据，n<=1000,m<=1000;
对于100%的数据，n<=100000,m<=100000.
