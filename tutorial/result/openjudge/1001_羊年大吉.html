1001:羊年大吉

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

你好，欢迎来到我的世界，我是一只羊，2015是我的名字，祝你羊年大吉！

在羊的世界里，每一只羊都被唯一的整数标记，并作为名字，比如我就叫2015。

在羊的世界里，羊与羊之间的血脉关系通过名字来了解，比如有两只羊，名字分别是X和Y，如果X和Y拥有不为1的公约数，那么意味着这两只羊有血缘关系。

有的时候，我很想知道和我有血缘关系的羊的数量，听说你编程很厉害，帮帮我吧！

输入
输入第一行包含一个整数T，表示测试数据的组数。
接下来的T行，每行一组测试数据，依次包含两个整数L和R，表示计算闭区间[L, R]当中跟我有血缘关系的其他羊的数量。
数据范围：
33%：1 <= T <= 1000，1 <= L <= R <= 10^3；
67%：1 <= T <= 1000，1 <= L <= R <= 10^9；
100%：1 <= T <= 1000，1 <= L <= R <= 10^18。
输出
对于每组测试数据，输出测试序号（从1开始），以及对应的答案。
样例输入
3
1 10
1 100
1 1000
样例输出
Case #1: 2
Case #2: 29
Case #3: 285
提示
测试点3个，各10分，共30分。
