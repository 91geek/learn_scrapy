1004:2014-09-26 拯救苹果

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
262144kB
描述

苹果树上长出了N 个苹果。为了简化问题，我们将空间看成二维的，地面为直线y=0，0 时
刻时第i 个苹果的坐标为(xi, yi)，速度为0。0 时刻后苹果由于受到引力产生大小为g 方向竖
直向下的加速度。Mr.Lin 初始时站在(0, 0)，体力值为0。为了获得额外的体力值，Mr.Lin
可以选择拯救一些苹果，但是他只能在地面上移动。假如一个苹果在时刻t（第t 秒末，第
t+1 秒初）落到地面上某点(x, 0)，而Mr.Lin 在此刻不在(x, 0)或者正在移动，它就会摔在地
上碎掉，瞬间灰飞烟灭；相反，如果Mr.Lin 此刻停在(x, 0)，苹果就得到了拯救，作为回报，
Mr.Lin 可以将苹果吃掉。吃苹果是可以增加体力值的。具体地，吃掉第i 个苹果获得的体力
值为Bi。不过，快速移动十分消耗体能。在T秒内移动Δx单位的距离耗费的体力值为kΔx /T。
由于Mr.Lin 拥有超能力，我们无需考虑速度问题（即Mr.Lin 的速度可以在0 时间内达到oo，
这不科学！），并且他的体力值是可以“透支”的（即他某时刻的体力值可以为负，这也不科
学！）
请求出Mr.Lin 最后体力值的最大值。
注：所有长度单位均为米，加速度单位米/(秒^2)

输入
第一行一个正整数N 和两个正实数g,k 分别表示苹果个数、苹果所受加速度大小即体力消耗
系数。
接下来N 行，每行三个实数xi, yi, Bi 描述苹果，含义如上所述。
输出
输出一行一个浮点数表示答案四舍五入至小数点后一位的结果。
样例输入
1 1.0 1.0
0.0 1.0 1.0
样例输出
1.0
提示
数据约束
对于 20%的数据，N <= 20。
对于 60%的数据，N <= 100
对于 100%的数据，1 <= N <= 1000,0 <= k, Bi, g <= 100,|xi| <=10^9
