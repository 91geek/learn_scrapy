1025:划分羊群

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
65536kB
描述

某羊群的主人拥有2N只羊，依次标号为0，1，2，...，2N-1。

他同时拥有两个牧场，不妨记为A和B，他希望将羊群划分到两个牧场，假设A牧场的羊数量为NA，B牧场的羊数量为NB，显然：

NA + NB = 2N，其中：NA>= 0, NB>= 0, N > 0。

记牧场A中的羊原始标号从小到大依次为：M1，M2，...，MNA，已知有：

M1 ^ M2 ^ ... ^ MNA = 0，其中：^ 表示异或运算。

试问满足条件的<M1，M2，...，MNA>有多少组？

输入
第一行包含一个整数T，表示测试数据的数量；
接下来的T行，每行包含一个整数N，表示羊的数量是2^N。
数据范围：
50%：1 <= T <= 5，1 <= N <= 5；
100%：1 <= T <= 10^4，1 <= N <= 10^18。
输出
对于每组测试数据，输出测试序号（从1开始）以及对应的结果，这个结果可能很大，因此，你只需要输出它模1000000007 （即10^9 + 7）的值即可。
样例输入
2
1
2
样例输出
Case #1: 2
Case #2: 4
提示
当N = 1时，共有2组，即：{Ø，<0>}；
当N = 2时，共有4组，即：{Ø，<0>，<1, 2, 3>，<0, 1, 2, 3>}；
测试点2个，各25分，共50分。
