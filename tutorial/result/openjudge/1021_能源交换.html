1021:能源交换

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
256000kB
描述
能源危机是一个迫在眉睫的问题。因此addspop想要拯救世界。他的目标是让每一块手机蓄电池都有等量的电能。初始时，每块蓄电池有各自的电能：第i块电池有a_i单位的电能。电能可以在不同的蓄电池间相互传递。假设蓄电池A和B之间进行电能传递（由A传给B），那么在x个单位时间里（x不一定是整数），A的电能将减少x个单位，由于传递时有k%的能量损耗，所以B只能接收到x-x*k%的能量。
addspop想要使所有蓄电池的电能都为同一个值，请你帮他计算这个值最大可以为多少。
输入
第一行，两个整数n,k (1 ≤ n ≤ 10000, 0 ≤ k ≤ 99)。n是蓄电池的数量，k见上述描述。
第二行n个数，依次表示每一块电池初识时的电量。a1, a2, ... , an(0 ≤ ai ≤ 1000, 1 ≤ i ≤ n)
输出
只有一行，进行能量传递后，蓄电池中相等电量的最大值。保留9位小数。
样例输入
第一组:
3 50
4 2 1
第二组:
2 90
1 11
样例输出
第一组:
2.000000000
第二组:
1.909090909
