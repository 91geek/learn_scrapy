描述

        这是一个简单的游戏，如图：

    给定9个方块上下左右的数字，要求将这些方块不旋转地排列成一个3*3的矩阵，使得上下或者左右相邻的方块的对应数字是一样的。给定初始的9个方块上的数字，请你判断是否可能完成任务。


输入
输入第一行为一个数T，下面有T组测试数据。
对于每组测试数据有9行，每行4个一位数字，以空格隔开，表示上、右、下、左的方块的数字。
输出
对于每组数据，输出一个数，YES表示能够完成任务，NO表示不行。
样例输入
1
0 1 3 1
3 0 0 2
3 1 3 1
3 3 0 1
3 1 3 1
0 1 8 6
8 2 0 3
0 6 3 0
8 1 8 1
样例输出
YES
提示
对于50%的数据，数据组数T<=10。
对于100%的数据，T<=100。
