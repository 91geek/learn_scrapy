B:二叉树的简单遍历

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
1000ms
内存限制:
128000kB
描述

给出一个n个节点的二叉树，请求出二叉树的前序遍历，中序遍历和后序遍历。0<n<=26

输入
第一位一个整数，表示二叉树有n个节点，以下n行，每行第一个为一个大写字母表示节点，后面为两整数，第一个表示左儿子序号，第二个表示右儿子序号，如果该序号为0表示没有
输出
共三行，第一行为二叉树的前序遍历，第二行为中序遍历，第三行为后序遍历
样例输入
7
F 2 3
C 4 5
E 0 6
A 0 0
D 7 0
G 0 0
B 0 0
样例输出
FCADBEG
ACBDFEG
ABDCGEF
