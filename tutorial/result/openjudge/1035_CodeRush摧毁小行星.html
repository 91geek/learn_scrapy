1035:CodeRush摧毁小行星

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
256000kB
描述

CodeRush想要驾驶他的飞船穿过一个N x N方格(1 <= N <= 500)的危险小行星地带.每个方格区域里面有K个小行星(1 <= K <= 10,000),它们都正好的落在了方格内的格点上.

幸运的是,CodeRush拥有一间强大的武器,每次一束激光可以摧毁指定一行或者一列上所有的小行星.这武器比较昂贵,所以他希望尽可能地节省使用.给出每个小行星的坐标,算出CodeRush摧毁所有小行星之后所使用的最小次数.

输入
输入文件planet.in。
第1行:两个整数N和K,由一个空格隔开
第2..K+1行:每一行都包括了两个由空格隔开的整数R和C(1 <= R, C <= N)分表表示小行星的横向和纵向坐标.
输出
输出文件planet.out仅包含一行，射击的最小次数。
样例输入
3 4
1 1
1 3
2 2
3 2
样例输出
2
提示
【样例解释】
下图表示了输入数据,”X”代表小行星,”.”代表空白:
X.X
.X.
.X.
CodeRush可以横向攻击第一列,摧毁了(1,1)和(1,3),然后纵向攻击第二列,摧毁了(2,2)和(3,2).
