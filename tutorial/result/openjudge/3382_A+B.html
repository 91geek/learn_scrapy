3382:A+B

总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
65536kB
描述
输入两个整数A 和 B，计算 A + B 。
输入
第1 行：2个整数 A 和 B (-2^31<= A， B <=2^31)
输出
第1行：1个整数，即 A + B 的值
样例输入
1 2
样例输出
3
提示
问: 数据输入和输出采用什么方式？
答: 你提交程序时必须采用 从标准输入读取数据，向标准输出写入数据的方式。 例如，你可以在C程序中使用 “scanf”，或者在C++程序中用 “cin”，或者在PASCAL程序中用“read”从标准输入中读取数据；在C程序中使用 “printf”，或者在C++程序中用 “cout”，或者在PASCAL程序中用“write”向标准输出写入数据。

你的程序不能向标准输出写入任何多余的数据，否则将会得到“答案错误 (Wrong Answer)”的结果。

你的程序不能从任何文件中输入/输出数据，否则将会得到“运行时错误(Runtime Error)”或者“答案错误 (Wrong Answer)”的结果。

以下是解决本题的C++(G++)样例程序:
#include
using namespace std;
int main()
{
int a,b;
cin >> a >> b;
cout << a+b << endl;
return 0;
}

注意：在G++/GCC程序中，main()函数的返回值类型必须是“整型(int)”,否则将得到“编译错误(compile error)”。

下面是这道题的一个C或者GCC的程序。
#include
int main()
{
int a,b;
scanf("%d %d",&a, &b);
printf("%d\n",a+b);
return 0;
}
