1003:等比数列前n项和

描述

陈大爷给zhx出了一道数论题。现在陈大爷让zhx对首项为1公比为k的等比数列的前n项求和， zhx不会写高精所以要陈大爷出简单一点，告诉zhx把答案对p取模。万万没想到的是，p有可能不是一个质数，但zhx只会算质数的取模，所以陈大爷只好把这道题扔给了你。

输入
一行三个整数n,k,p分别表示项数、公比和模。
输出
一行个整数，代表答案对p取模的值。
样例输入
3 3 233
样例输出
13
提示
等比数列求和公式为(k^n-1)/(k-1)。
对于 30%的数据， 1≤n,k≤100。
对于 60%的数据， 1≤n,k≤10^9，p是质数。
对于 80%的数据， 1≤n,k≤10^9，p=a×b，其中a,b均为质数。
对于 100%的数据， 1≤n,k,p≤10^9。
