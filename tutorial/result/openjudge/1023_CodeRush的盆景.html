1023:CodeRush的盆景

  • 查看
  • 提交
  • 统计
  • 提问
总时间限制:
10000ms
单个测试点时间限制:
1000ms
内存限制:
256000kB
描述
CodeRush的草场上有一条河，河中间有一些石头，编号为1到N，CodeRush想在这些石头上摆放盆景。每一块石头都有一个美学值，当在这一块石头上摆放盆景，就可以获得这块石头对应的美学值。
摆放规则是：如果你现在在第i块石头上摆放了盆景，那么下一步你只能在第 i+L 块到第 i+R 块石头之间的选择一格摆放。

为了便于描述，我们假设第0块石头在河岸上（为了不影响最终结果，把第0块石头的美学值设为0）。

假设CodeRush已经在第0块石头上摆放了盆景，请你继续帮他往下摆放直到达到河对岸（只要下一步的位置编号大于N就算到达对岸），并使美学值之和最大。
输入
第1行：3个正整数N, L, R
第2行：N+1个整数，第i个数表示编号为i-1的石头的美学值
输出
共一行，一个整数，表示最大的美学值的和。
样例输入
5 2 3
0 12 3 11 7 -2
样例输出
11
提示
对于60%的数据：N <= 10,000
对于100%的数据：N <= 200,000 ， -1,000 <= A[i] <= 1,000 ，1 <= L <= R <= N
保证最终结果在longint范围内。
