Description

crf是刷子王国的国王，他拥有操纵永远与须臾的能力，并且将这个能力用在打crossfire上。但有一天他被小学生血虐了，他才发现他已经失去了这个能力，于是他决定将算法竞赛定为全国唯一的比赛项目，因为他认为即使失去能力，自己的刷题能力也是世界最强的，并且宣布如果有谁能做出他出的五道难题，他就可以成为刷子王国的新国王。你，做好准备了吗？

给定一棵有根树（根为1号结点），每个节点初始值为0，有三种操作，分别为将以某个结点为根的子树中所有节点的值全部设置为1，将某个结点到根节点的路径上所有结点（包括该结点和根节点）的值全部设置为0，询问某个结点的值

Input

第一行为一个数n(1 <= n <= 500000)，表示结点个数，编号为1~n

接下来n-1行，每行两个整数x y，表示结点x和结点y之间有一条边

接下来一行为一个数q(1 <= q <= 500000)，表示有q次操作

接下来q行，每行两个整数x y(1 <= x <= 3 1 <= y <= n)

x=1表示将以y为根节点的子树中所有结点的值全部设置为1

x=2表示将结点y到跟的路径上所有结点的值全部设置为0

x=3表示询问结点y的权值

Output

对于每个x=3的操作，输出一行，一个数，代表该结点的值(0或1)
