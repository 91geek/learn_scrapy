描述

勇敢与鲁莽是好朋友。


有一天鲁莽要挑战勇敢的智商，给勇敢出一道题目：已知一个序列，f[1] = 1, f[2] = 1, f[i] = f[i - 1] + f[i - 2](i >= 3)，要问这个序列中任意2个数的整除关系。


现在请你化身为勇敢，迎接鲁莽的挑战。

输入
输入包含多组数据，每一组数据占一行，包含2个整数i，j，满足(3 <= i <= j <= 1000000)。当i = j = 0时，输入结束。
输出
对于每一行输入，包含一行输出，0表示f[j]不能被f[i]整除，1表示f[j]可以被f[i]整除。
样例输入
3 7
4 8
5 5
0 0
样例输出
0
1
1
