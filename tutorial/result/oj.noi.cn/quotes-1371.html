1371. 植物大战僵尸 (Standard IO)

具体限制 

题目描述

输入

输出
输出文件pvz.out仅包含一个整数，表示可以获得的最大能源收入。注意，你也可以选择不进行任何攻击，这样能源收入为0。

样例输入

 
3 2
10 0
20 0
-10 0
-5 1 0 0
100 1 2 1
100 0

样例输出

 
25

约20%的数据满足1 ≤ N, M ≤ 5；
约40%的数据满足1 ≤ N, M ≤ 10；
约100%的数据满足1 ≤ N ≤ 20，1 ≤ M ≤ 30，-10000 ≤ Score ≤ 10000
