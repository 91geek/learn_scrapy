1146. 最长不下降子序列 (Standard IO)

时间限制: 1000 ms  空间限制: 262144 KB  具体限制 

题目描述
给定一个n个数的序列A，问最长的不下降子序列长度为多少。对于一个序列p1< p2< … < pk,其为不下降子序列需要满足：A[p1] <= A[p2] <= A[p3] <= … <= A[pk]。

输入
第一行一个整数n。
接下来一行n个整数，表示A[i]。

输出
一个整数，表示最长的不下降子序列的长度。

样例输入

 
7
1 7 3 5 9 4 8

样例输出

 
4

数据范围限制
n <= 5000,|A[i]|<=1000000。
