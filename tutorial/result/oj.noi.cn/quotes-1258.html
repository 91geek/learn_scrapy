1258. 连通子数 (Standard IO)

具体限制 

题目描述
每个测试点时限：4秒          内存限制：1024MB

小M至上主义者小D今天心情很好，准备给小M准备一个圣诞礼物。
他购买了一个圣诞树，这个圣诞树是一个有n个点的树，点i有一个颜色。小D觉得如果某种颜色太多就不好看了，所以最多会有5个点有相同的颜色。
小M收到了礼物很高兴，但是她最近对计数问题很感兴趣，于是她有一些问题想要小D解决。每次小M会关心三种不同的颜色a，b，c，她想要知道这个圣诞树有多少个不同的非空联通子图满足里面颜色为a，b，c的点的个数分别为，，。由于数可能很大，输出关于对+7取模的余数就可以了。
小D当然轻松解决了这个问题，你虽然被为了狗粮，但你也想锻炼你的实力，你能解决这个问题吗？

输入
从标准输入读入数据。
第一行两个数n和Q，表示树的大小和询问的个数。第二行n个数，其中第i个表示，为第i个节点的颜色。接下来n-1行，每行两个数a和b，表示点a和点b之间有一条边，接下来Q行，其中每行有6个数a，，b，，c，，表示如题目中所述的询问。

输出
输出到标准输出。
对每个询问输出一行表示答案。

样例输入

 
5  3
1  2  3  1  2
1  2
2  3
3  4
4  5
1  1  2  1  3  1
1  0  2  1  3  1
1  1  2  1  3  0

样例输出

 
3
1
2

数据范围限制
对于每个$1 ≤ i ≤ n$，1 ≤  ≤ n。
对于每个询问我们有：0 ≤ ，， ≤ 5，1 ≤ a，b，c ≤ n以及a，b，c两两不同。
对于10%的询问，n，Q ≤ 10。
对于另10%的询问，n ≤ 15，Q ≤ 50。
对于另10%的询问，n，Q ≤ 1000。
对于另5%的询问，n，Q ≤ 50000，并且输入的树如果以点1为根，最大深度不超过35（点1的深度为0）。
对于另5%的询问，n，Q ≤ 100000，并且输入的树如果以点1为根，最大深度不超过35（点1的深度为0）。
对于另20%的询问，n，Q ≤ 100000，并且保证所有询问$（n_a+1）^2·（n_b+1）2·（n_c+1）2$  。
对于另40%的询问，n，Q ≤ 100000，并且保证所有询问$（n_a+1）（n_b+1）（n_c+1）$的和小于等于8·。
