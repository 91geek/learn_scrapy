1315. 文学 (Standard IO)

具体限制 

题目描述
每个测试点时限：2秒          内存限制：256MB

输入
第一行两个正整数n，p，分别表示“批量阅读”的方案数以及书的数量。
接下来n行，每行四个整数，其中第i行包含四个整数，，，，表示第i种“批量阅读”的方案。
接下来p行，每行两个整数，其中第i行包含两个整数，，表示第i本书的参数。

输出
一行一个整数，表示最少需要的时间。若无论如何也无法读完全部书籍，则输出-1。

样例输入

 
4 3
-1 0 0 10
-1 -1 -1 2
-1 1 -1 2
-1 0 -1 1
0 2
0 -2
1 0

样例输出

 
4
