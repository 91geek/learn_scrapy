100003. 上楼梯 (Standard IO)

时间限制: 1 ms  空间限制: 131072 KB  具体限制 

题目描述
楼梯有N级台阶，上楼可以一步上一级台阶，也可以一步上两级台阶。编一程序，计算共有多少种不同的走法。

输入
一行，一个整数n（1 <= n <= 30），表示台阶的级数

输出
一个整数，走法的种数

样例输入

 
3

样例输出

 
3
方案1：走三次，每次一步
方案2：先走一级，再走两级
方案3：先走两级，再走一次
