100270. 方格染色 (Standard IO)

具体限制 

题目描述
每个测试点时限：3秒          内存限制：256M

输入
输入的第一行包含三个整数n, m和k，分别代表表格的行数、列数和已被染色的方格数目。
之后的k行描述已被染色的方格。其中第i行包含三个整数, 和，分别代表第i个已被染色的方格的行编号、列编号和颜色。为1表示方格被染成红色，为0表示方格被染成蓝色。

输出
输出一个整数，表示可能的染色方案数目W模得到的值。（也就是说，如果W大于等于，则输出W被除所得的余数）。

样例输入

 
3 4 3
2 2 1
1 2 0
2 3 1

样例输出

 
8

数据范围限制
对于20%的测试数据，n, m ≤ 5，k ≤ 5；
对于50%的测试数据，n, m ≤ 5000，k ≤ 25；
对于所有的测试数据，2 ≤ n, m ≤ ，0 ≤ k ≤ ，1 ≤  ≤ n，1 ≤  ≤ m。
