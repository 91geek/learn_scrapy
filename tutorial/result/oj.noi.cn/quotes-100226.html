100226. 志愿者招募 (Standard IO)

具体限制 

题目描述
每个测试点时限：2s          内存限制：128M

申奥成功后，布布经过不懈努力，终于成为奥组委下属公司人力资源部门的主管。布布刚上任就遇到了一个难题：为即将启动的奥运新项目招募一批短期志愿者。经过估算，这个项目需要N天才能完成，其中第i天至少需要A 个人。布布通过了解得知，一共有M 类志愿者可以招募。其中第i 类可以从第S 天工作到第T 天，招募费用是每人C 元。新官上任三把火，为了出色地完成自己的工作，布布希望用尽量少的费用招募足够的志愿者，但这并不是他的特长！于是布布找到了你，希望你帮他设计一种最优的招募方案。

输入
输入文件employee.in 的第一行包含两个整数N, M，表示完成项目的天数和可以招募的志愿者的种类。
接下来的一行中包含N 个非负整数，表示每天至少需要的志愿者人数。
接下来的M 行中每行包含三个整数S, T, C，含义如上文所述。为了方便起见，我们可以认为每类志愿者的数量都是无限多的。

输出
输入文件 employee.out 中仅包含一个整数，表示你所设计的最优方案的总费用。

样例输入

 
3 3
2 3 4
1 2 2
2 3 5
3 3 2

样例输出

 
14

数据范围限制
30%的数据中，1 ≤ N, M ≤ 10，1 ≤ A ≤ 10；
100%的数据中，1 ≤ N ≤ 1000，1 ≤ M ≤ 10000，题目中其他所涉及的数据均不超过2 -1。
