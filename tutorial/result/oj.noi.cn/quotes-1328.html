1328. 华容道 (Standard IO)

具体限制 

题目描述
每个测试点时限：1秒          内存限制：128MB

输入

输出
输出文件为puzzle.out。输出有q行，每行包含1个整数，表示每次游戏所需要的最少时间，如果每次游戏无法完成目标则输出-1。

样例输入

 
3 4 2
0 1 1 1
0 1 1 0
0 1 0 0
3 2 1 2 2 2
1 2 2 2 3 2

样例输出

 
2
-1

数据范围限制
对于30%的数据，1≤n，m≤10，q=1；
对于60%的数据，1≤n，m≤30，q≤10；
对于100%的数据，1≤n，m≤30，q≤500。
