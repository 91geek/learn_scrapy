1313. Sum (Standard IO)

具体限制 

题目描述

输入
第一行一个数T，表示有T组测试数据。接下来T行，每行两个正整数n，r。

输出
输出T行，每行一个正整数表示答案。

样例输入

 
3
3 5
3 6
3 7

样例输出

 
3
1
-1

数据范围限制
对于30%的数据，满足n≤，r≤，T≤10；对于60%的数据，满足n≤，r≤，T≤；对于100%的数据，满足n≤，r≤，T≤。
