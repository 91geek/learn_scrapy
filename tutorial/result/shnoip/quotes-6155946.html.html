Crt单元

Posted on 2016-12-10 15:56 noipxu 阅读(...) 评论(...) 编辑 收藏

一、调用单元
例：uses crt;

二、清屏
例：clrscr;

三、移动光标
例：gotoxy(a,b);
其中a表示列号，b表示行号

四、清行
例：clreol;
清除光标所在行光标上及以后的所有字符

五、文本颜色
例：textcolor(blue);
颜色表:(参数可以是颜色名 也可以是一个整型)
Black = 0; Blue = 1; Green = 2; Cyan = 3; Red = 4; Magenta = 5; Brown = 6; LightGray = 7; DarkGray = 8; LightBlue = 9; LightGreen = 10; LightCyan = 11; LightRed = 12; LightMagenta = 13; Yellow = 14; White = 15; Blink = 128;

六、延迟
例：delay(1000);用于让程序暂停几秒，括号里参数的单位是毫秒

七、其它函数
Window(x1,y1,x2,y2); 划定一块窗口区域

TextBackground(Color); 确定背景颜色

WhereX WhereY 返回光标位置(x表示列，y表示行)
